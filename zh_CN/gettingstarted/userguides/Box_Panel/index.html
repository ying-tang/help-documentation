      开始使用 Box Panel

      Box Panel 界面提供了一个统一视图，可用于查看 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud Local 环境中部署的资产。通过其自助服务，您可以轻松地创建支持问题单、将中心认证用于登录和用户许可权、监控各种功能以及查看发票。

      本用户指南为您提供了全方位的概述，帮助您开始使用 Box Panel 来监视和控制 IBM Bluemix Cloud 环境。本文档适用于 Box Panel V3.8.3 和更高版本。

      目录

      • 登录

      • 浏览 Box Panel 仪表板

      • 帮助文档

      登录

      要开始使用 Box Panel，请进行登录。

      1. 转至 URL https://boxpanel.bluebox.net 以定向至登录页面。
      2. 使用 IBM Bluemix Private Cloud 支持团队提供的登录凭证来登录到您的 Box Panel 帐户。如果您是次要联系人，并且还没有访问权，那么可由管理员将您添加进来。请参阅管理用户和项目部分，以获取更多详细信息。
      3. 如果您还没有 Box Panel 帐户，请参阅计费部分，以了解有关如何获取帐户的指示信息。
      4. 登录后，系统会自动将您定向至 Box Panel 仪表板。

      从 Lockbox 获取凭证

      最初，您会通过 Box Panel Lockbox 收到私有云（OpenStack 集群）的 cloud_admin 凭证。您的凭证应类似如下：

      要开始操作，请首先浏览至 Lockbox 中提供的 URL。然后，可以使用您指定的用户名和您指定的密码来登录。

      浏览 Box Panel 仪表板

      在 Box Panel 仪表板中，可以在一个屏幕上查看所有 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud Local 云主机的统一资产视图。

      该仪表板顶部显示当前正在使用的基础架构以及您的主机列表。

      在右侧显示与帐户汇总相关的其他详细信息，包括 LockBox 消息、口头密码、带宽配额和备份配额。

      在右上方，可以看到以下选项卡和菜单选项。

      选项卡：

      • 支持：查看现有的问题单和创建新的问题单。请参阅获取支持，以获取更多信息。
      • 交谈：与 IBM Bluemix Private Cloud 支持人员实时交谈。请参阅获取支持中的“使用交谈功能与 Bluemix Private Cloud 支持人员交谈”部分。
      • Webmail：使用 https://mail.blueboxgrid.com/ 上托管的 Webmail。
      • 用户：查看或更新您的个人档案，或者更改您的密码。

      菜单：

      • 切换帐户：如果您有权访问多个 IBM Bluemix Private Cloud 云主机，那么可以使用此选项切换至其他帐户。
      • 帐户：查看帐户联系人、LockBox 消息、计费信息、上报过程以及帐户信息。
      • 服务：查看您帐户可用的所有 Blue Box Cloud 和管理型托管服务，包括服务器、云镜像、DNS、带宽图、带宽数据和 New Relic。选项可能因客户类型而异。
      • 文档：查看帮助文档。请参阅帮助文档，以获取更多信息。

      管理联系人

      要在 Box Panel 中查看和更新客户联系人信息，请执行以下步骤：

      • 登录到 Box Panel。
      • 从帐户菜单中，选择帐户联系人以打开“帐户联系人”页面。

      • “已授权联系人”部分下列出有权访问您帐户的所有联系人。将列出每个人员的姓名和电子邮件地址以及三种联系人角色（请注意，一个联系人可具有其中的一种或多种联系人角色）：

        • 管理 (A) 联系人可以订购或移除服务，还可以管理其他联系人
        • 计费 (B) 联系人可以查看和支付发票
        • 技术 (T) 联系人可以开具问题单并联系 Bluemix Private Cloud 支持团队以获取帮助
        • 主要 (P) 联系人。这是唯一一个保留帐户管理控制权的联系人。只有 IBM Bluemix Private Cloud 支持人员才可以更改主要联系人。

      要添加新的联系人，请单击添加联系人按钮。此时会显示添加联系人表单，您可以在此表单中输入联系人的姓名、电子邮件地址、电话号码以及联系人将具有的角色。完成后，单击“保存详细信息”按钮以添加该联系人。

      要移除现有联系人，请单击“已授权联系人”列表中此联系人所在行中的 X。

      请注意，无法移除主要联系人。请联系 IBM Bluemix Private Cloud 支持人员以让其协助您修改主要联系人。

      使用云镜像

      只有至少具有一个云且为主要或技术客户联系人的客户才能使用 Box Panel 中的云镜像页面。如果您取得用户资格，那么服务导航中将包含可访问此页面的云镜像链接。

      每个镜像都显示为一张卡片。这些卡片由操作系统进行分组。

      要下载云镜像：

      1. 将鼠标指针悬停在与要下载的镜像相关联的卡片上。
      2. 单击此卡片。此次单击将打开一个模态窗口，其中包含指向镜像以及校验和的下载链接。
      3. 单击镜像 URL 左侧的复制按钮。此次单击会将该 URL 添加到剪贴板中。
      4. 也可以使用 OpenStack API 来完成此过程。有关指示信息，请参阅云镜像的主文档

      查找帮助文档

      在 Box Panel 仪表板中，可在导航栏上看到一个称为文档的链接（最右侧的链接）。如果选择该链接，会显示一个下拉菜单，其中包含 OpenStack、云对象存储器或管理型托管选项。要查看 IBM Bluemix Private Cloud 的“客户帮助”文档，请选择 OpenStack。此时会显示“帮助”页面。您可以按类别浏览此文档，也可以使用搜索栏进行搜索。通过将鼠标指针悬停在某个图标上，即可看到该主题领域中的文章列表。

      获取支持

      注：要立即获取支持，请拨打电话 1-800-613-4305 或发送电子邮件至 support@bluebox.net 来联系 Bluemix Private Cloud 支持人员。

      在大多数情况下，您需要开具支持问题单来获取支持。通过使用问题单，可保留一条记录来记载问题及其解决方案，以及与客户支持团队的交谈会话的日志。

      要创建问题单：

      步骤 1：单击支持页面中的新建问题单。

      步骤 2：在“模态”窗口中，您的姓名将显示为问题单创建者。

      步骤 3：选择类别（支持或计费）。确认将有权访问该问题单的用户列表。

      步骤 4：将其他用户复制到抄送其他用户字段中，使其知晓该问题单并跟踪其状态。

      步骤 5：为该问题单添加描述性主题。

      步骤 6：收集所有有用信息并将其添加到描述部分中，例如：

      • 开始发生问题的日期和时间
      • 该问题出现的频率（永久或经常性、一天中的特定时间等）
      • 用于重现该问题的步骤
      • 受影响的服务器名称
      • 受影响的数据中心
      • 返回的任何错误消息
      • 适用的服务器日志
      • 任何其他有用信息

      步骤 7：在描述部分右下方包含附件（如日志文件）。

      步骤 8：如果部署针对的是新云，请选中部署框。

      注：对于非紧急问题，可以跳过步骤 9 和步骤 10。

      步骤 9：如果问题紧急，可以选中将此问题单开具为紧急问题单框以上报此问题单。此次单击会提示您确认在问题不属于紧急情况（包括停运情况在内）时同意支付咨询费用。

      步骤 10：如果您想要将问题单开具为紧急问题单，并且同意支付非紧急项的咨询费用，请选中我同意框。

      步骤 11：最后，单击创建问题单框。

      使用支持问题单

      在创建问题单后，通过在仪表板中单击该支持问题单，可随时查看与该问题单相关联的文本。问题单的文本中可能包含对其他问题单的引用，以及交谈历史记录和任何与所选问题单相关的通信内容。在右上角会突出显示支持问题单的状态。状态指示符包括 OPEN、PENDING COMPLETION、PARKED 等。

      在屏幕左侧的面板中，您可以选择相应的选项以新建另一个支持问题单、预订所选问题单（如果想要关注其状态）、查看问题单，以及查看您帐户上的共享用户提交的所有问题单。

      要查看所选问题单的订户列表，请单击订户链接。

      如果想要与 Bluemix Private Cloud 支持团队的某位专家就此问题单的主题进行书面对话，可以单击绿色的发布回复按钮，也可以单击交谈来使用实时交谈功能。所有实时交谈会话都会记录下来，并以抄本形式添加到问题单中。

      使用交谈功能与 Bluemix Private Cloud 支持人员交谈

      1. 单击仪表板中的交谈选项卡。此操作会自动将您定向至用于和支持人员交谈的聊天室。
      2. 在这里，您可以在聊天室的左侧查看加入对话的支持团队成员。
      3. 支持团队成员加入后，您可以在交谈栏中编写和发送消息。如果在交谈期间需要输入新行，请按 Ctrl/Command + Enter。
      4. 如果需要上报支持问题，请致电 1-800-613-4305 转 1。

      注：每个客户聊天室都是私人聊天室。这意味着只有您客户帐户中的联系人和 Bluemix Private Cloud 支持人员才可以访问您的抄本。将不会持久存储敏感数据（如信用卡号或私钥）。

      管理计费

      要查看您的合同和每月计费报告，请单击 Box Panel 仪表板中的帐户选项卡，然后单击计费选项卡。

      要获取更多的计费支持，请联系 IBM 支持团队，我们很乐意为您提供帮助。
