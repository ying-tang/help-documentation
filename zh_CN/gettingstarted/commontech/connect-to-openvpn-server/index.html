      连接到 OpenVPN 服务器

      • by Jason Kennedy ● January 7th, 2016

      • Tags:

        • getting started

        • 入门

        • openVPN

        • 连接

      既然您已在 OpenStack 集群上创建了 OpenVPN 服务器，现在就进行连接：

      请记住，不论如何命名 .ovpn 文件，都将调用此连接。在我们的示例中，因为文件名为 Client1.ovpn，连接将命名为 Client1。

      Windows

      安装

      可在 OpenVPN 下载页面上找到适用于 Windows 的 OpenVPN 客户端应用程序。为您的 Windows 版本选择相应的安装程序版本。

      注：OpenVPN 需要管理特权来执行安装。

      在安装 OpenVPN 后，将统一的 DigitalOcean.ovpn 概要文件复制到：

      C:\Program Files\OpenVPN\config

      在启动 OpenVPN 时，它将自动查看概要文件并提供该概要文件。

      每次在使用 OpenVPN 时都必须以管理员身份运行，甚至是通过管理帐户进行使用。要在每次使用 VPN 时执行此操作而不必单击鼠标右键并选择以管理员身份运行，您可以预先设置此值，但是必须通过管理帐户来完成。此限制还意味着标准用户将需要输入管理员密码才能使用 OpenVPN。另一方面，标准用户无法正确地连接到服务器，除非客户端上的 OpenVPN 具有管理权限，因此需要提升的特权。

      要将 OpenVPN 应用程序设置为始终以管理员身份运行，请右键单击其快捷方式图标并转至属性。在兼容性选项卡的底部，单击更改所有用户的设置按钮。在新窗口中，选中以管理员身份运行此程序。

      连接

      每次启动 OpenVPN GUI 时，Windows 都将询问是否想要允许程序对计算机执行更改。单击是。启动 OpenVPN 客户端应用程序仅会将 applet 放入系统托盘中，因此可根据需要连接 VPN 和断开连接；它不会实际进行 VPN 连接。

      在启动 OpenVPN 后，通过转至系统托盘 applet 并右键单击 OpenVPN applet 图标来启动连接。这将打开上下文菜单。选择菜单顶部的 Client1（也就是我们的 Client1.ovpn 概要文件），然后选择连接。

      在建立连接时将打开一个状态窗口，其中显示日志输出，并且在连接客户端后将显示一条消息。

      以相同方式断开 VPN 连接：转至系统托盘 applet，右键单击 OpenVPN applet 图标，选择客户端概要文件，然后单击断开连接。

      OS X

      安装

      Tunnelblick 是针对 Mac OS X 的免费开源 OpenVPN 客户端。您可以从 Tunnelblick 下载页面下载最新的磁盘镜像。双击下载的 .dmg 文件并遵循提示执行安装。

      在安装过程即将结束时，Tunnelblick 将询问是否有任何配置文件。可简单地回答否，让 Tunnelblick 完成操作。打开查找程序窗口并双击 DigitalOcean.ovpn。Tunnelblick 将安装客户端概要文件。需要管理特权。

      连接

      通过双击应用程序文件夹中的 Tunnelblick，启动 Tunnelblick。在启动 Tunnelblick 后，屏幕右上角的菜单栏中将有一个用于控制连接的 Tunnelblick 图标。单击该图标，然后单击连接菜单项以启动 VPN 连接。选择 Client1 连接。

      iOS

      安装

      从 iTunes App Store 中搜索并安装 OpenVPN Connect（官方 iOS OpenVPN 客户端应用程序）。要将 iOS 客户端概要文件传输到设备，请直接将其连接到计算机。

      此处概述如何使用 iTunes 完成传输。打开计算机上的 iTunes，单击 iPhone > 应用。向下滚动至文件共享部分的底部，然后单击 OpenVPN 应用。右侧的空白窗口 OpenVPN 文档用于共享文件。将 .ovpn 文件拖至 OpenVPN 文档窗口。

      现在，启动 iPhone 上的 OpenVPN 应用。将会出现一个通知，表明新概要文件已准备好进行导入。点击绿色加号以将其导入。

      连接

      OpenVPN 现已准备好使用新的概要文件。通过将连接按钮滑至打开位置开始连接。通过将相同按钮滑至关闭位置断开连接。

      注：设置下面的 VPN 开关无法用于连接到 VPN。如果尝试，那么将收到通知，表明只能使用 OpenVPN 应用进行连接。

      Android

      安装

      打开 Google Play 商店。搜索并安装 Android OpenVPN Connect（官方 Android OpenVPN 客户端应用程序）。

      可通过 USB 将 Android 设备连接到计算机并复制文件来传输 .ovpn 概要文件。或者，如果您有 SD 读卡器，那么可卸下设备的 SD 卡，将概要文件复制到该卡上，然后将卡插回到 Android 设备中。

      启动 OpenVPN 应用并点击菜单以导入概要文件。

      然后，浏览至保存的概要文件的位置（屏幕快照使用 /sdcard/Download/）并选择文件。该应用将记录概要文件已导入。

      连接

      要进行连接，只需点击连接按钮。将询问您是否信任 OpenVPN 应用程序。选择确定以启动连接。要断开 VPN 连接，请返回至 OpenVPN 应用并选择 *断开连接。

      测试 VPN 连接

      在安装所有项之后，执行简单检查以确认一切正常工作。无需启用 VPN 连接，只需打开浏览器并转至 DNSLeakTest。

      该站点将返回您的因特网服务提供商分配的 IP 地址，这也是您对外显示的 IP 地址。要通过相同的 Web 站点检查 DNS 设置，请单击 Extended Test，这将告知您正在使用的 DNS 服务器。

      现在，将 OpenVPN 客户端连接到 Droplet VPN 并刷新浏览器。现在应该会显示完全不同的 VPN 服务器 IP 地址。这就是现在您对外显示的样子。另外，DNSLeakTest Extended Test 将检查您的 DNS 设置，并确认您目前正在使用 VPN 推送的 DNS 解析器。

      祝贺您！现在，您可以安全地浏览因特网，同时保护您的身份、位置和流量，远离窥探者和审查者。
