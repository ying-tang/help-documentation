      从 Horizon 仪表板中启动云主机（OpenStack Newton 发行版）

      • by Ying Tang ● March 31, 2017

      • Tags:

        • horizon

        • horizon

        • 云主机

        • 启动

        • newton

        • rhosp

      步骤

      1. 登录 Horizon 仪表板。
      2. 在项目主题下，展开计算，然后单击云主机。如果您目前具有云主机，可以查看云主机列表及其详细信息，如名称、IP 地址和状态。有关管理云主机的信息，请参阅如何管理云主机。
      3. 单击 Horizon 仪表板右上角的启动云主机。将显示启动云主机对话框。

      浏览以下选项卡并提供值，然后单击对话框左下方的启动云主机。

      填充“启动云主机”对话框中的值

      “详细信息”选项卡

      值 描述
      云主机名称 向虚拟机分配名称。您在此分配的名称会成为服务器的初始主机名。在创建服务器时，不保证服务器名称是唯一的，因此可能会存在主机名相同的两个云主机。
      可用性区域 默认情况下，该值将设置为云提供者提供的可用性区域 (AZ)，例如，compute_standard。
      计数 要启动多个云主机，请输入大于 1 的值。默认值为 1。

      要继续，请单击下一步或单击源选项卡。

      “源”选项卡

      选择引导源字段提供了以下选项：

      选项 描述
      镜像  
      从可用表中的私有镜像和公共镜像列表中进行选择，然后单击加号 (+) 将其移动到已分配。（可选）您可以单击创建新云硬盘上的是来启用更多云硬盘创建选项。若已经使用此云主机创建新云硬盘，如果您想在云主机终止时删除此云硬盘，可以单击删除云主机时删除云硬盘选项上的是。  
      云主机快照 从可用表中的快照进行选择，然后单击加号 (+) 将其移动到已分配。
      云硬盘 从可用表中的云硬盘进行选择，然后单击加号 (+) 将其移动到已分配。（可选）要在云主机终止时删除此云硬盘，您可以单击删除云主机时删除云硬盘选项上的是。
      云硬盘快照 从可用表中的云硬盘快照进行选择，然后单击加号 (+) 将其移动到已分配。（可选）要在云主机终止时删除此云硬盘，您可以单击删除云主机时删除云硬盘选项上的是。

      注：默认情况下已设置镜像选项，因此您在引导之前不必选择。如果您使用镜像选项，可以从 IBM 提供的公共镜像中进行选择，也可以上传和使用自己的镜像。有关如何向 IBM Bluemix Private Cloud 上传镜像的信息，请参阅 IBM Bluemix Private Cloud 提供的云镜像。

      “云主机类型”选项卡

      从云主机类型列表中进行选择，然后单击加号 (+) 将其移动到已分配。

      “网络”选项卡

      从网络列表中进行选择，然后单击加号 (+) 将其移动到已分配。您可以将多个网络分配到此云主机。

      “网络端口”选项卡

      如果您不指定任何网络端口，那么创建和引导云主机时将向其分配默认网络端口。

      有时，您可能需要提前通过 API 创建端口。创建端口并将其与您在网络选项卡中选择的一个网络进行关联后，您可以在可用表中查看此端口。您可以单击加号 (+) 将其移动到已分配。

      “安全组”选项卡

      从安全组列表中进行选择，然后单击加号 (+) 将其移动到已分配。

      “密钥对”选项卡

      指定密钥对。具有以下选项：

      • 从已存储的密钥对列表中进行选择，然后单击加号 (+) 将所选项移动到已分配。
      • 单击导入密钥对以导入您自己的密钥对。复制并粘贴公钥，然后单击导入密钥对。

      • 单击创建密钥对以创建和存储新密钥对。指定密钥对名称，然后单击创建密钥对。

        将提示您将新的密钥对保存到本地主机。

      “配置”选项卡

      连接到云主机的自定义脚本可以在启动云主机时执行特定操作。您可以直接将脚本输入到自定义脚本字段。

      对于 Linux 云主机：默认用户名为 ibmcloud。要通过控制台和 SSH 启用密码认证，请为自定义脚本使用以下脚本数据（使用相关密码来替换 <YOUR_PASSWORD>）：

      #cloud-config password: <YOUR_PASSWORD> chpasswd: { expire: False } ssh_pwauth: True

      对于 Windows 云主机：要设置初始用户名和密码，请为自定义脚本使用以下脚本数据（使用相关用户名和密码来替换 <YOUR_USERNAME> 和 <YOUR_PASSWORD>）。

      rem cmd
net user <YOUR_USERNAME> <YOUR_PASSWORD> /logonpasswordchg:yes /add /y
net localgroup administrators <YOUR_USERNAME> /add

      如果您的浏览器支持 HTML5 File API，可以单击浏览从文件中加载脚本。脚本大小不应超过 16 Kb。

      在磁盘分区下拉菜单中，您可以选择自动来调整磁盘大小，将其设置为单个分区，也可以选择手动以在磁盘上创建多个分区。

      如果您要将元数据写入特定配置驱动器，请选择配置驱动器复选框，然后在元数据选项卡中指定元数据。云主机引导时，将连接到该配置驱动器以访问您的元数据。

      “服务器组”选项卡

      服务器组可用于控制一组服务器（云主机）的关联性和反关联性调度策略。

      有时，您可能需要提前通过 API 创建服务器组。创建服务器组后，您将能够在可用表中查看此服务器组。您可以单击加号 (+) 将其移动到已分配。

      “调度程序提示”选项卡

      调度程序提示可用于将其他放置相关信息传递到 Nova 调度程序。例如，您可以使用调度程序提示来确保将云主机调度到特定服务器组中的主机上。您还可以使用 JSON 格式构造自定义调度程序提示。有关调度机制和受支持提示类型的更多详细信息，请参阅计算调度程序。

      “元数据”选项卡

      Glance 提供了可定义键和值对的通用 API，这些键和值对用于标记不同类型资源（镜像、工件、云硬盘、云主机类型、聚集等）的元数据。如果您选择的镜像中包含预定义元数据，您将在可用元数据列中看到此元数据。

      您可以在自定义字段中创建其他键和值对，然后单击加号（+）将其移动到现有元数据列表。例如，自定义元数据是您要传递到应用程序的参数，因此可以随云主机一起安装。

      有关 Glance 元数据的更多信息，请参阅元数据定义概念和使用 Glance 的元数据定义目录公共 API。

      如何连接到云主机

      注：请确保为通过 ssh 或 rdp 连接云主机定义了相应的安全组规则。

      1. 登录 IBM Bluemix Private Cloud Horizon 仪表板。

      2. 展开项目面板，然后展开左侧导航区域中的计算选项卡，并单击云主机。

      3. 在运行的云主机列表中找到您的云主机，记下其 IP 地址，例如 192.168.100.237。

      4. 通过指定 IP 地址来创建与云主机的连接。

      连接到 Linux 云主机

      通过 SSH 或 VNC 连接到云主机。IBM 提供的 Linux 镜像的默认 userid 为 ibmcloud。连接到 Linux 云主机后，可以使用 sudo 命令执行通常需要 root 用户访问权的命令。Sudo 配置为不需要默认 userid 的密码。

      • 通过首选的 SSH 客户端，采用配置期间指定的密钥对的 SSH 私钥，与 VM 云主机的正确 IP 地址建立连接。

          ssh -i <path_of_Your_SSH_private_key_file> <userid>@<ip_of_instance>

      • 在 IBM Bluemix Private Cloud Horizon 仪表板中，通过在云主机的操作列中单击下拉列表中的控制台来查看云主机的 VNC 控制台。VNC 控制台使用 HTTPS 进行连接。

      注：要以 ibmcloud 用户身份通过控制台登录新的 Linux 云主机，必须在配置期间先前已在自定义脚本中指定密码。

      连接到 Windows 云主机

      通过远程桌面或 VNC 连接到云主机。您将需要使用 VM 配置期间在自定义脚本中指定的 userid 和 password。首次登录时，会显示一个提示，要求您更改密码。请更改密码。

      • 使用您首选的远程桌面客户端进行连接。
      • 在 IBM Bluemix Private Cloud Horizon 仪表板中，通过在云主机的操作列中单击下拉列表中的控制台来查看云主机的 VNC 控制台。VNC 控制台使用 HTTPS 进行连接。

      如何管理云主机

      在项目主题下，展开计算，然后单击云主机。

      仪表板会显示当前项目中的云主机列表。如下所示，在“操作”列中，下拉列表显示了您的选项：

      • 创建快照：创建云主机的快照。*
      • 关联浮动 IP：将可用浮动 IP 关联到云主机的一个端口。
      • 连接接口：将接口连接到网络。
      • 拆离接口：移除您为云主机创建的端口。
      • 编辑云主机：更改云主机名称，或上传安全组。
      • 更新元数据：更新您为云主机指定的自定义元数据。
      • 编辑安全组：移除现有安全组，或分配新的安全组。
      • 控制台：查看云主机的 VNC 控制台。
      • 查看日志：查看云主机日志。
      • 暂停云主机：暂停云主机并将其状态保存在内存中。此选项将保留资源（虚拟内核及内存）。
      • 暂挂云主机：暂挂云主机并将其状态保存到磁盘。此选项将释放资源（虚拟内核及内存）。
      • 搁置云主机：搁置镜像并关闭云主机。
      • 调整云主机大小：更新云主机的云主机类型设置。
      • 锁定云主机：锁定云主机。
      • 解锁云主机：解锁云主机。
      • 软重启云主机：执行正常关闭并重新启动云主机。
      • 硬重启云主机：执行等效于服务器电源复位的操作。
      • 关闭云主机：切断云主机电源。
      • 重建云主机：使用另一个镜像重建云主机。
      • 删除云主机：删除此云主机。

      单击需要应用的操作选项。
