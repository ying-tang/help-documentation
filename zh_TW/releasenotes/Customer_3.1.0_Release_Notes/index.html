      IBM Bluemix Private Cloud 客戶

      版本注意事項 3.1.0

      2016 年 10 月 14 日

      本文件提供 3.1.0 版 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud（本端）供應項目的基本說明。您可以在每一個小節中找到其他說明文件的鏈結。

      最新版本的摘要

      3.1.0 版的 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud（本端）包含一些新功能和效能改善。它仍然是以 OpenStack Mitaka 版本為基礎。

      3.1.0 版的每一個 IBM Bluemix Private Cloud 都有下列新增功能：

      1.) 聯合 Keystone–客戶身分提供者 (IDP)：現在客戶可以利用其現有的「身分基礎架構」聯合通訊協定（例如「安全主張標記語言 (SAML)」或 OpenIDConnect），來使用它們自己的身分目錄（例如 LDAP、RADIUS 及 Active Directory）中所儲存的身分。由客戶管理的單一認證可用來管理多個 OpenStack 雲端。

      以下是一些相關文件：

      • K2K 聯合
      • SAML 聯合
      • OIDC 聯合

      2.) 儲存空間儲存區：有不同儲存空間效能需求的客戶，現在可以最佳方式將工作負載導向至相同雲端中的高效能 SSD 和具有成本效益的混合式儲存空間儲存區。用於新部署的 OSD 節點數下限仍為 3 個，若為混合式擴充，則必須新增至少 3 個節點。之後，您就可以擴充儲存空間，一次一個節點。

      以下是一些相關文件：

      • 支援的磁區類型
      • 從 Horizon 建立磁區
      • 從指令行建立磁區

      3.) 儲存空間效能改善：Ceph 部署已升級至已啟用 Jemalloc 的 Jewel 版本，以提供效能改善。

      IBM Bluemix Private Cloud 映像檔更新項目

      IBM Bluemix Private Cloud 映像檔的更新：我們可用的「雲端映像檔」檔案庫已更新。雲端映像檔的使用手冊含有最新資訊。

      IBM Bluemix Private Cloud 3.1.0 功能（在 IBM Cloud 資料中心）：

      • 「IBM Cloud 資料中心」的 Bluemix Private Cloud 現在符合 ISO27001 及 270018 標準，外加 SOC 2 和 3。

      IBM Bluemix Private Cloud 3.1.0（本端）功能：

      • 支援 Lenovo：除了現行 Cisco UCS 平台之外，現在還支援 Lenovo 策劃的資料清單。
      • 支援 Bluemix 本端系統：「Bluemix 本端系統」現在支援 IBM Bluemix Private Cloud（本端）的「私有雲即服務 (PCaaS)」。

      此版本的已知限制

      在 Horizon 上，聯合使用者有一個頁面可以變更其密碼。嘗試變更密碼時發生錯誤，因為其身分資訊未儲存。（問題報告 #236958）

      在 Horizon 上，透過群組授與 project_admin 存取權的使用者，在身分->專案中可能看不到任何使用者。（問題報告 #237038）

      如需 LBaaS 限制的特定資訊，請參閱 LBaaS 第 2 版的限制。

      如需其他使用者說明文件，請參閱位於下列網址且可隨時從線上取得的主要客戶說明頁面：help.blueboxcloud.com。

      對於 IBM Bluemix Private Cloud 的新使用者，IBM Bluemix Private Cloud 2.0 版以及更新版本的一般產品概觀可供使用。
