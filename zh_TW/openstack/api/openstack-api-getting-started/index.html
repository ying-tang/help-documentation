      開始使用 OpenStack API

      • by Eric French ● November 16th, 2015

      • Tags:

        • api

        • api

        • openstack

      IBM Bluemix Private Cloud 附有 API 端點，可用來執行所有主要的 OpenStack 服務。以下說明如何設定系統以存取這些服務：

      在 macOS 上：

      安裝 Python

      附註：macOS El Capitan 所採用的機制可以避免損壞作業系統檔案。/System/Library/Frameworks/Python.framework/Versions/2.7/share 是其中一個受保護的位置。一般使用者沒有理由在該處放置或寫入任何檔案。因此，我們並不建議您使用 sudo pip 來安裝 Python。

      請改為將 Python 套件（例如 IPython）安裝到本端環境中使用者的起始檔案夾。最簡單的方式，就是建立一個虛擬環境，啟動它，然後在該虛擬環境中執行 pip。

      範例：

      設定本端 stackrc 檔案

      stackrc 檔案定義的環境變數可讓您的 Nova 用戶端連接至 Nova API：

      建立您自己的 stackrc，並將 OS_USERNAME、OS_PROJECT_NAME、OS_TENANT_NAME、OS_USER_DOMAIN_NAME、OS_PROJECT_DOMAIN_NAME、OS_CACERT 及 OS_PASSWORD 欄位取代為您自己的值。

      附註：如果您不放心將密碼儲存在這個檔案，請將密碼欄位保留空白；系統會在取用 stackrc 檔案後要求您輸入密碼。

      也可以從 Horizon 下載 OpenStack RC 檔。請登入 Horizon，導覽至存取與安全頁面，然後選取 API 存取標籤。

      按一下下載 OpenStack RC 檔按鈕，然後視需要修改該檔案。

      如何使用 cURL 查詢 OpenStack API

      問：如何透過第 3 版 API 取得 OpenStack 記號？

      答：您可以使用 cURL 指令：

      如需相關資訊，請參閱：

      這裡鏈結的 API 手冊含有使用 cURL 取得伺服器清單的範例。

      每一個服務都有各自用於 API 存取的埠。如果您登入 Web GUI 並移至存取與安全 => API 存取標籤，就可查看每個埠所對應的服務。

      若要查詢任何其他 API 服務（Nova、Neutron、Cinder 等），您必須先取得鑑別服務 (Keystone) 的記號。在該記號到期前，您可以重複使用該記號，以發出所需數目的 API 呼叫。

      例如，9696 是 Neutron，但若要取得記號（您嘗試執行的指令），則需要使用埠 5000 的 Keystone 鑑別要求。

      因此，步驟如下：

      1. 使用 HTTP POST（而非 GET，因為您需要 POST 您的 $OS_USERNAME 及 $OS_PASSWORD, $OS_TENANT_NAME，才能擷取記號），從 $OS_AUTH_URL/tokens 提取記號。回應將包含記號和不同服務的型錄。

      2. 擷取該記號，然後檢查記號的到期日是否為未來。

      3. 從傳回的型錄中，擷取您要對其發出 API 呼叫之服務的 URL。在您的案例中，您要在型錄中尋找類型為 network 的項目。項目名稱應為 neutron。

      4. 當您對該 URL 發出不同的 HTTP GET 時，請併入該記號作為 X-Auth-Token 標頭。您可以執行 neutron --debug security-group-list 以瞭解 POST 看起來的形態，它應該類似下列內容：

      curl -g -i --cacert "/etc/ssl/certs/ca-certificates.crt" \
-X GET https://example.openstack.blueboxgrid.com:9696/v2.0/security-groups.json \
-H "User-Agent: python-neutronclient" \
-H "Accept: application/json" \
-H "X-Auth-Token: {SHA1}11111a1111aaa1a111aaaaaa1111aa1a111a1111"
