      連接至 OpenVPN 伺服器

      • by Jason Kennedy ● January 7th, 2016

      • Tags:

        • getting started

        • 開始使用

        • openVPN

        • 連接

      既然您已在 OpenStack 叢集上建立了 OpenVPN 伺服器，現在就來連接它：

      請記住，此連線將以 .ovpn 檔案的命名作為其名稱。在我們的範例中，因為此檔案是命名為 Client1.ovpn，所以此連線將命名為 Client1。

      Windows

      安裝

      您可以在 OpenVPN 下載頁面上找到適用於 Windows 的 OpenVPN 用戶端應用程式。請選擇您的 Windows 版本所適用的安裝程式版本。

      附註：OpenVPN 需要管理專用權才能安裝。

      安裝 OpenVPN 之後，將統一的 DigitalOcean.ovpn 設定檔複製到下列位置：

      C:\Program Files\OpenVPN\config

      當您啟動 OpenVPN 時，它將自動查看設定檔並使它成為可用狀態。

      每次使用 OpenVPN 時，都必須以管理者身分執行，即使是由管理帳戶使用也一樣。若要達成此目的，但不想要每次使用 VPN 時都按一下滑鼠右鍵選取以系統管理員身分執行，您可以預設此值，但必須從管理帳戶完成此動作。這項限制也表示標準使用者需要輸入管理者的密碼才能使用 OpenVPN。在另一方面，除非在用戶端上的 OpenVPN 具備管理權，否則標準使用者無法適當地連接至伺服器，因此需要較高的專用權。

      若要設定 OpenVPN 應用程式一律以管理者身分執行，請用滑鼠右鍵按一下其捷徑圖示，然後移至內容。在相容性標籤底端，按一下按鈕來變更所有使用者的設定。在新視窗中，勾選以系統管理員的身分執行此程式。

      連接

      每次啟動 OpenVPN GUI 時，Windows 會詢問您是否要容許程式變更您的電腦。按一下是。啟動 OpenVPN 用戶端應用程式只會將 Applet 放入系統匣，因此可視需要與 VPN 連接和中斷連接；它不會實際建立 VPN 連線。

      OpenVPN 啟動之後，請移至系統匣 Applet，並用滑鼠右鍵按一下 OpenVPN Applet 圖示，以起始連線。這會開啟快速功能表。選取功能表頂端的 Client1（亦即我們的 Client1.ovpn 設定檔），然後選擇連接。

      建立連線時，會開啟顯示日誌輸出的狀態視窗，而在連接用戶端之後，會顯示一則訊息。

      以相同方式與 VPN 中斷連接：移至系統匣 Applet，用滑鼠右鍵按一下 OpenVPN Applet 圖示，選取用戶端設定檔，然後按一下中斷連接。

      OS X

      安裝

      Tunnelblick 是一種適用於 Mac OS X 的免費開放程式碼 OpenVPN 用戶端。您可以從 Tunnelblick 下載頁面下載最新的磁碟映像檔。按兩下已下載的 .dmg 檔案，並遵循提示來進行安裝。

      在安裝程序快要結束時，Tunnelblick 會詢問您是否有任何配置檔。回答否而讓 Tunnelblick 完成會比較簡單。開啟搜尋器視窗，然後按兩下 DigitalOcean.ovpn。Tunnelblick 將安裝用戶端設定檔。需要有管理專用權。

      連接

      在應用程式資料夾中按兩下 Tunnelblick 來啟動 Tunnelblick。啟動 Tunnelblick 之後，畫面右上角的功能表列中會有一個 Tunnelblick 圖示，用來控制連線。按一下此圖示，然後按一下連接功能表項目，以起始 VPN 連線。選取 Client1 連線。

      iOS

      安裝

      從 iTunes App Store 搜尋並安裝 OpenVPN Connect，這是官方的 iOS OpenVPN 用戶端應用程式。若要將 iOS 用戶端設定檔傳送至該裝置，請將它直接連接到電腦。

      這裡將概述如何使用 iTunes 完成傳送。在電腦上開啟 iTunes，然後按一下 iPhone > 應用程式。向下捲動到底端的檔案共用區段，然後按一下 OpenVPN 應用程式。右邊的空白視窗 OpenVPN 文件用於共用檔案。請將 .ovpn 檔案拖曳到 OpenVPN 文件視窗。

      現在，請在 iPhone 上啟動 OpenVPN 應用程式。會有一個通知，指出新的設定檔可以開始匯入。點選綠色加號即可匯入它。

      連接

      OpenVPN 現在可以開始使用新的設定檔。請將連接按鈕移至開啟位置，以啟動連線。將相同的按鈕移至關閉，即可中斷連線。

      附註：設定下的 VPN 交換器無法用來連接至 VPN。如果您嘗試這麼做，就會收到通知，指出您只能使用 OpenVPN 應用程式進行連接。

      Android

      安裝

      開啟 Google Play 商店。搜尋並安裝 Android OpenVPN Connect，這是官方的 Android OpenVPN 用戶端應用程式。

      可以透過 USB 將 Android 裝置連接至您的電腦，並將檔案複製過來，以傳送 .ovpn 設定檔。或者，如果您有 SD 讀卡機，則可以取出該裝置的 SD 卡，將設定檔複製到其中，然後將 SD 卡插回到 Android 裝置。

      啟動 OpenVPN 應用程式，並點選功能表來匯入設定檔。

      然後，導覽至已儲存設定檔的位置（擷取畫面使用 /sdcard/Download/），並選取該檔案。應用程式會記錄已匯入該設定檔。

      連接

      若要連接，只要點選連接按鈕即可。系統會詢問您是否信任 OpenVPN 應用程式。請選擇確定以起始連線。若要中斷與 VPN 的連接，請回到 OpenVPN 應用程式，然後選擇 *中斷連接。

      測試 VPN 連線

      安裝所有項目之後，進行簡易檢查，以確認一切正常運作。在沒有啟用 VPN 連線的情況下，開啟瀏覽器並移至 DNSLeakTest。

      此網站會傳回由網際網路服務供應商指派的 IP 位址，您在世界其他地區使用它來代表自己。若要透過相同的網站檢查您的 DNS 設定，請按一下延伸測試，它將告訴您所使用的是哪些 DNS 伺服器。

      現在，將 OpenVPN 用戶端連接至 Droplet 的 VPN，並重新整理瀏覽器。現在應該會出現您的 VPN 伺服器完全不同的 IP 位址。現在，這是您在全世界用來代表自己的方式。同樣地，DNSLeakTest 延伸測試會檢查 DNS 設定，並確認您現在使用由 VPN 推送的 DNS 解析器。

      恭喜！您現在可以安全地遍訪網際網路，同時您的身分、位置及資料傳輸受到保護，可以避開探查者和審查者。
