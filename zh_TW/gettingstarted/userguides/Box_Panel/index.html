      開始使用 Box Panel

      Box Panel 介面為您提供在 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud（本端）環境中部署的資產單一合併視圖。它的自助式存取方式可讓您輕鬆建立支援問題單、對登入和使用者權限運用中央鑑別、監視各種功能，以及檢視您的發票。

      本使用手冊提供端對端概觀，因此您可以開始使用 Box Panel 來監視及控制 IBM Bluemix 雲端環境。本文件適用於 Box Panel 3.8.3 版以及更新版本。

      目錄

      登入

      若要開始使用 Box Panel，請登入。

      1. 移至 URL https://boxpanel.bluebox.net，以導向至登入頁面。
      2. 使用 IBM Bluemix Private Cloud 支援團隊提供的登入認證來登入 Box Panel 帳戶。如果您是次要聯絡人，而且還沒有存取權，則管理者可以將您新增進來。如需詳細資料，請參閱管理使用者和專案一節。
      3. 如果您還沒有 Box Panel 帳戶，請參閱計費一節，以瞭解如何取得帳戶的相關指示。
      4. 登入之後，會將您自動導向至 Box Panel 儀表板。

      從鎖箱取得您的認證

      一開始，您將透過 Box Panel 鎖箱收到您的私有雲（OpenStack 叢集）的 cloud_admin 認證。您的認證如下所示：

      若要開始使用時，請先導覽至鎖箱中提供的 URL。然後，您可以使用您指定的使用者名稱和您指定的密碼登入。

      導覽 Box Panel 儀表板

      從 Box Panel 儀表板中，您可以在同一個畫面中看到您所有的 IBM Bluemix Private Cloud 和 IBM Bluemix Private Cloud（本端）執行實例的合併資產視圖。

      儀表板頂端顯示目前使用中的基礎架構，以及您的主機清單。

      右邊顯示與您的帳戶摘要相關的其他詳細資料，包括鎖箱訊息、口頭密碼、頻寬配額及備份配額。

      在右上方，您可以檢視下列標籤和功能表選項。

      標籤：

      • 支援：可檢視現有的問題單及建立新的問題單。如需相關資訊，請參閱取得支援。
      • 會談：可與 IBM Bluemix Private Cloud 支援人員進行即時會談。請參閱取得支援中的「使用會談功能與 Bluemix Private Cloud 支援中心會談」一節。
      • Web 郵件：可使用 https://mail.blueboxgrid.com/ 上代管的 Web 郵件。
      • 我自己：可檢視或更新您的設定檔，或變更您的密碼。

      功能表：

      • 切換帳戶：如果您有權存取多個 IBM Bluemix Private Cloud 執行實例，則可以使用此選項切換到另一個帳戶。
      • 帳戶：檢視您的帳戶聯絡人、鎖箱訊息、帳單資訊、呈報程序及帳戶資訊。
      • 服務：檢視您帳戶可用的所有 Blue Box 雲端及主機代管服務等服務，包括伺服器、雲端映像檔、DNS、頻寬圖形、頻寬資料以及 New Relic。不同類型的客戶可能有不同的選項。
      • 文件：檢視說明文件。如需相關資訊，請參閱說明文件。

      管理聯絡人

      若要在 Box Panel 中檢視及更新客戶聯絡人資訊，請執行下列步驟：

      • 登入 Box Panel。
      • 從帳戶功能表中選擇帳戶聯絡人，以開啟「帳戶聯絡人」頁面。

      • 已獲授權存取您帳戶的所有聯絡人會列在「授權聯絡人」區段下。會列出每位人員的姓名和電子郵件位址，以及三個聯絡人角色（請注意，一個聯絡人可以有一個或多個聯絡人角色）：

        • 管理 (A) 聯絡人可以訂購或移除服務，也可以管理其他聯絡人
        • 計費 (B) 聯絡人可以檢視及支付發票
        • 技術 (T) 聯絡人可以開啟問題單及聯絡 Bluemix Private Cloud 支援團隊以尋找協助
        • 主要 (P) 聯絡人。這位人員（只有一位）保有帳戶的管理控制權。「主要聯絡人」只能由 IBM Bluemix Private Cloud 支援人員加以變更。

      若要新增聯絡人，請按一下新增聯絡人按鈕。即會顯示新增聯絡人表單，可讓您輸入聯絡人的姓名、電子郵件位址、電話號碼，以及聯絡人應該屬於哪些角色。完成時，請按一下「儲存詳細資料」按鈕，以新增聯絡人。

      若要移除現有的聯絡人，請在「授權聯絡人」清單中具有其名稱的那一列按一下 X。

      請注意，您無法移除「主要聯絡人」。修改「主要聯絡人」時如需協助，請與 IBM Bluemix Private Cloud 支援中心聯絡。

      使用雲端映像檔

      Box Panel 中的雲端映像檔頁面可供至少有一個雲端的客戶，以及本身是主要或技術客戶聯絡人的客戶使用。如果您是合格的使用者，服務導覽會包括一個雲端映像檔鏈結讓您存取該頁面。

      每個映像檔都會顯示為一張卡。這些卡會依作業系統分組。

      若要下載雲端映像檔，請執行下列動作：

      1. 將滑鼠移至與您要下載的映像檔相關聯的卡片上。
      2. 按一下該卡片。這個按一下動作會開啟限制模式視窗，其中包含映像檔和總和檢查的下載鏈結。
      3. 按一下映像檔 URL 左側的複製按鈕。這個按一下動作會將 URL 新增到剪貼簿。
      4. 此程序也可以透過使用 OpenStack API 來達成。如需指示，請參閱關於雲端映像檔的主要文件

      尋找說明文件

      從 Box Panel 儀表板中，您會看到導覽列上有一個鏈結（最右邊的鏈結）稱為文件。如果您選取該鏈結，則會出現一個下拉功能表，其中含有 OpenStack、雲端物件儲存空間或主機代管服務等選項。若要檢視 IBM Bluemix Private Cloud 的客戶說明文件，請選取 OpenStack。即會顯示「說明」頁面。您可以依種類瀏覽說明文件，或使用搜尋列來搜尋。將滑鼠游標移到其中一個圖示上，您就可以查看該主題區域內的文章清單。

      取得支援

      附註：如需立即支援，您可以撥打 1-800-613-4305 來聯絡「Bluemix Private Cloud 支援中心」，或傳送電子郵件到 support@bluebox.net 與我們聯絡。

      在大部分情況下，您可以開啟支援問題單來取得支援。透過使用問題單，您會獲得一份記錄，內含您的問題以及解決該問題所採取的解決方案，並完整包含您與客戶支援團隊之間會談階段作業的日誌。

      若要建立問題單，請執行下列動作：

      步驟 1. 按一下支援頁面中的新建問題單。

      步驟 2. 在限制模式視窗中，您的名稱應該會顯示為問題單建立者。

      步驟 3. 選取種類（支援或計費）。請驗證使用者清單應該可以存取您的問題單。

      步驟 4. 將副本抄送給副本抄送給其他使用者欄位中的其他使用者，讓他們知道該問題單並追蹤其狀態。

      步驟 5. 在問題單中新增敘述性主旨。

      步驟 6. 收集所有有用資訊，並新增至說明區段，例如：

      • 問題的開始日期和時間
      • 問題的發生頻率（永久或固定於一天的特定時間發生等等）。
      • 要重現問題時所採取的步驟
      • 受影響的伺服器名稱
      • 受影響的資料中心
      • 傳回的任何錯誤訊息
      • 適用的伺服器日誌
      • 任何其他有用資訊

      步驟 7. 直接在說明區段右下方包括附件，例如日誌檔。

      步驟 8. 如果您的部署是針對新的雲端，請勾選部署方框。

      附註：如果不是緊急狀況，可跳過步驟 9 和 10。

      步驟 9. 如果此問題很緊急，您可以勾選開啟此問題單並標示為緊急方框來呈報問題單。這個按一下動作會提示您確認：您同意在問題不嚴重（嚴重項目包括中斷服務的狀況）時支付諮詢費用。

      步驟 10. 如果您想要開啟問題單並標示為緊急，且同意支付不嚴重項目的諮詢費用，請勾選我同意方框。

      步驟 11. 最後，按一下建立問題單方框。

      使用支援問題單

      建立問題單之後，您可以隨時按一下儀表板中的這個支援問題單，來查看與該問題單相關聯的文字。此問題單的文字可能包括對其他問題單的參照，以及會談歷程和所選取問題單的任何相關書信。右上角會強調顯示您的支援問題單的狀態。狀態指示器包括 OPEN、PENDING COMPLETION、PARKED 等等。

      從螢幕左側的畫面中，您可以選取選項來建立另一個新的支援問題單、訂閱已選取的問題單（如果您想要追蹤其狀態）、檢視您的問題單，以及檢視由您帳戶的共用使用者提交的所有問題單。

      若要檢視所選取問題單的訂閱者清單，請按一下訂閱鏈結。

      若要與我們其中一位 Bluemix Private Cloud 支援團隊專家針對此問題單的主題進行書面交談，您可以按一下綠色貼文回覆按鈕，也可以按一下會談來使用即時會談功能。所有即時會談階段作業都會記錄下來，並新增至問題單成為記錄。

      使用會談功能與 Bluemix Private Cloud 支援中心會談

      1. 按一下「儀表板」中的會談標籤。這個動作會將您自動導向至有支援人員的會談室。
      2. 從這裡，您可以在會談室左側檢視加入交談的支援人員成員。
      3. 當支援成員加入之後，您可以從會談列編寫及傳送訊息。如果在會談期間您需要輸入新的一行，請按 Control/Command + Enter 鍵。
      4. 如果您需呈報支援問題，請撥打 1-800-613-4305 分機 1。

      附註：每一位客戶的會談室都是專用的。這表示只有客戶帳戶內的聯絡人以及 Bluemix Private Cloud 支援人員才可以存取您的記錄。不會持續保存機密資料（例如信用卡號碼或私密金鑰）。

      管理帳單

      若要檢視合約和每月帳單報告，請按一下 Box Panel 儀表板中的帳戶標籤，然後按一下計費標籤。

      如需進一步的計費支援，請聯絡 IBM 支援團隊，我們將非常樂意協助您。
