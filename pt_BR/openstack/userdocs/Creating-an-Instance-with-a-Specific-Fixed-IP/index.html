      Creating an Instance with a Specific Fixed IP

      • by Blue Box Support ● January 2016

      • Tags:

        • openstack

        • instance

        • fixed IP

      Pode ser necessário criar uma instância com um endereço IP privado, específico (também conhecido por IP Fixo) a partir da sub-rede. Para fazer isso, deve-se criar manualmente uma porta neutron. Este artigo esboça as etapas que atingem esse objetivo. Será necessário utilizar a API ou as ferramentas de linha de comandos para executar estas etapas.

      Pré-requisitos

      1. Se for usar as ferramentas da linha de comandos (CLI), certifique-se de ter ambas as ferramentas instaladas. As instruções de instalação para ambas as ferramentas podem ser encontradas em nossa base de conhecimento: Introdução à API do OpenStack
      • NeutronClient
      • NovaClient
