      Einführung in Box Panel

      In der grafischen Benutzerschnittstelle von Box Panel werden in einer einzigen konsolidierten Ansicht die in Ihren IBM Bluemix Private Cloud- und IBM Bluemix Private Cloud Local-Umgebungen bereitgestellten Assets angezeigt. Mit dem Self-Service-Zugriff können Sie ohne großen Aufwand Support-Tickets erstellen, die zentrale Authentifizierungsfunktion für Anmeldungen und Benutzerberechtigungen nutzen, verschiedene Funktionen überwachen und Ihre Rechnungen anzeigen.

      Dieses Benutzerhandbuch gibt Ihnen einen umfassenden Überblick, der es Ihnen ermöglicht, Box Panel für die Überwachung und Steuerung Ihrer IBM Bluemix Cloud-Umgebung zu verwenden. Dieses Dokument gilt für Box Panel Version 3.8.3 und höher.

      Inhaltsverzeichnis

      • Anmelden

      Anmelden

      Melden Sie sich an, um mit Box Panel zu arbeiten.

      1. Geben Sie die URL https://boxpanel.bluebox.net ein, um die Anmeldeseite aufzurufen.
      2. Melden Sie sich bei Ihrem Box Panel-Konto mit den Anmeldeberechtigungsnachweisen an, die Sie vom Support-Team von IBM Bluemix Private Cloud erhalten haben. Wenn Sie ein so genannter sekundärer Kontakt sind und noch keine Zugriffsberechtigung haben, können Sie von einem Administrator für den Zugriff berechtigt werden. Ausführliche Informationen finden Sie im Abschnitt Benutzer und Projekte verwalten.
      3. Wenn Sie kein Box Panel-Konto haben, lesen Sie den Abschnitt Abrechnung, um Informationen darüber zu erhalten, wie Sie ein Konto erstellen können.
      4. Nach der Anmeldung werden Sie automatisch zum Box Panel-Dashboard umgeleitet.

      Berechtigungsnachweise von Lockbox abrufen

      Sie erhalten zunächst die cloud_admin-Berechtigungsnachweise für Ihre private Cloud (OpenStack-Cluster) über die Box Panel-Lockbox. Ihre Berechtigungsnachweise sollte ähnlich dem folgenden Beispiel aussehen:

      Navigieren Sie zunächst zu der URL, die in Ihrer Lockbox bereitgestellt wurde. Geben Sie anschließend die Werte für Ihren_angegebenen_Benutzernamen und Ihr_angegebenes_Kennwort ein, um sich anzumelden.

      Im Box Panel-Dashboard navigieren

      Im Box Panel-Dashboard werden in einer konsolidierten Assetansicht alle Ihre IBM Bluemix Private Cloud- und IBM Bluemix Private Cloud Local-Instanzen in einer einzigen Anzeige angezeigt.

      Im oberen Bereich des Dashboards wird die momentan verwendete Infrastruktur zusammen mit einer Liste von Hosts angezeigt.

      Rechts werden zusätzliche Details im Zusammenhang mit Ihrer Kontoübersicht, einschließlich Lockbox-Nachrichten, Verbales Kennwort, Bandbreitenkontingent und Sicherungskontingent, angezeigt.

      In der rechten oberen Ecke können Sie die folgenden Register und Menüoptionen anzeigen.

      Registerkarten:

      • Support: Zeigen Sie hier vorhandene Tickets an und erstellen Sie neue Tickets. Weitere Informationen finden Sie unter Support anfordern.
      • Chat: Im Live-Chat können Sie mit Mitarbeitern des Support-Teams von IBM Bluemix Private Cloud sprechen. Weitere Informationen finden Sie im Abschnitt “Chat-Feature zum Chatten mit dem Bluemix Private Cloud-Support verwenden” unter Support anfordern.
      • Webmail: Verwenden Sie den Webmail-Service, der über den Host https://mail.blueboxgrid.com/ bereitgestellt wird.
      • Mein Konto: Hier können Sie Ihr Profil anzeigen und aktualisieren sowie Ihr Kennwort ändern.

      Menüs:

      • Konto wechseln: Wenn Sie auf mehrere IBM Bluemix Private Cloud-Instanzen zugreifen können, verwenden Sie diese Option, um zu einem anderen Konto zu wechseln.
      • Konten: Zeigen Sie hier die Kontakte zu Ihrem Konto, Lockbox-Nachrichten, Abrechnungsinformationen, Eskalationsverfahren und Kontoinformationen an.
      • Services: Zeigen Sie hier alle Blue Box Cloud- und Managed Hosting-Services an, die für Ihr Konto zur Verfügung stehen, einschließlich Server, Cloud-Images, DNS, Bandbreitendiagramme, Bandbreitendaten und New Relic. Die Optionen können je nach Kunde variieren.
      • Dokumente: Zeigen Sie hier die Hilfedokumentation an. Weitere Informationen finden Sie in der Hilfedokumentation.

      Kontakte verwalten

      Führen Sie die folgenden Schritte aus, um Ihre Kundenkontaktdaten in Box Panel anzuzeigen und zu aktualisieren:

      • Melden Sie sich bei Box Panel an.
      • Wählen Sie im Menü Konto den Eintrag Kontokontakte aus, um die Seite mit den Kontokontakten zu öffnen.

      • Im Abschnitt “Berechtigte Kontakte” werden alle Kontakte aufgelistet, die für den Zugriff auf Ihr Konto berechtigt sind. In der Liste wird der Name und die E-Mail-Adresse jeder Person zusammen mit drei Kontaktrollen genannt (Beachten Sie, dass ein Kontakt mehr als eine dieser Kontaktrollen haben kann.):

        • Kontakte mit der Rolle Admin (A) können Services bestellen und entfernen sowie andere Kontakte verwalten.
        • Kontakte mit der Rolle Abrechnung (B) können Rechnungen anzeigen und bezahlen.
        • Kontakte mit der Rolle Technisch (T) können Tickets erstellen und sich an das Bluemix Private Cloud-Support-Team wenden, um Unterstützung anzufordern.
        • Kontakt mit der Rolle Primär (P). Diese Person (und nur diese eine Person) hat die Verwaltungskontrolle über das Konto. Der primäre Kontakt kann nur von einem Mitarbeiter des IBM Bluemix Private Cloud-Support-Teams geändert werden.

      Wenn Sie einen neuen Kontakt hinzufügen möchten, klicken Sie auf die Schaltfläche Kontakt hinzufügen. Das Formular Kontakt hinzufügen wird angezeigt, in dem Sie den Namen und die E-Mail-Adresse des Kontakts mit Telefonnummer sowie die Rollen angeben können, die dem Kontakt zugewiesen werden sollen. Wenn Sie mit der Eingabe fertig sind, klicken Sie auf die Schaltfläche “Details speichern”, um den Kontakt hinzuzufügen.

      Wenn Sie einen vorhandenen Kontakt entfernen möchten, klicken Sie in der Liste “Berechtigte Kontakte” auf das “X” in der Zeile neben dem entsprechenden Namen.

      Beachten Sie, dass der primäre Kontakt nicht entfernt werden kann. Wenden Sie sich an das IBM Bluemix Private Cloud-Support-Team, um Unterstützung beim Ändern des primären Kontakts zu erhalten.

      Mit Cloud-Images arbeiten

      Die Seite Cloud-Images in Box Panel steht Kunden zur Verfügung, die mindestens eine Cloud verwenden und entweder ein primärer oder ein technischer Kundenkontakt sind. Wenn Ihre Benutzerrolle diese Bedingungen erfüllt, enthält die Navigation unter Services auch einen Link, über den Sie auf die Seite Cloud-Images zugreifen können.

      Jedes Image wird als Karte angezeigt. Die Karten sind nach Betriebssystem gruppiert.

      **Gehen Sie wie folgt vor, um ein Cloud-Image herunterzuladen: **

      1. Bewegen Sie die Maus über die Karte, die dem Image zugeordnet ist, das Sie herunterladen möchten.
      2. Klicken Sie auf die Karte. Auf diese Weise wird ein Modalfenster mit Download-Links zum Image und der Kontrollsumme geöffnet.
      3. Klicken Sie auf die Schaltfläche Kopieren links neben der Image-URL. Mit diesem Klick wird die URL Ihrer Zwischenablage hinzugefügt.
      4. Sie können diese Prozedur auch mithilfe der OpenStack-APIs durchführen. Anweisungen hierzu finden Sie im Hauptdokument zu Cloud-Images.

      Hilfedokumentation suchen

      Im Dashboard von Box Panel wird in der Navigationsleiste ganz rechts ein Link mit dem Namen Dokumente angezeigt. Wenn Sie diesen Link auswählen, wird ein Dropdown-Menü mit den folgenden Optionen angezeigt: OpenStack, Cloudobjektspeicher und Managed Hosting. Wenn Sie die IBM Bluemix Private Cloud-Hilfedokumentation für Kunden anzeigen möchten, wählen Sie den Eintrag OpenStack aus. Daraufhin werden die Hilfetextseiten angezeigt. Sie können die Dokumentation nach Kategorie durchsuchen, oder Sie verwenden die Suchleiste zum Suchen. Wenn Sie die Maus über die Symbole bewegen, wird eine Liste der Artikel zu diesem Themenbereich angezeigt.

      Support anfordern

      Anmerkung: Wenn Sie sofort Unterstützung benötigen, rufen Sie den Bluemix Private Cloud-Support unter der Nummer 1-800-613-4305 (USA) an, oder senden Sie eine E-Mail an support@bluebox.net.

      In der Regel sollten Sie ein Support-Ticket öffnen, um Unterstützung anzufordern. Wenn Sie ein Ticket öffnen, wird ein Datensatz zu dem Problem erstellt, der die Lösungen, die zur Behebung Ihres Problems geführt haben, und die Protokolle der Chat-Sitzungen mit den Mitarbeitern des Support-Teams enthält.

      **Gehen Sie wie folgt vor, um ein Ticket zu öffnen: **

      Schritt 1. Klicken Sie auf der Seite Support auf Neues Ticket.

      Schritt 2. Im Modalfenster sollte Ihr Name als Ticketersteller erscheinen.

      Schritt 3. Wählen Sie eine Kategorie (Support oder Abrechnung) aus. Gehen Sie die Liste der Benutzer durch, die Zugriff auf Ihr Ticket haben sollen.

      Schritt 4. Kopieren Sie zusätzliche Benutzer in das Feld Weitere Benutzer auf CC setzen, damit diese über das Ticket informiert werden und den zugehörigen Status überwachen können.

      Schritt 5. Geben Sie einen beschreibenden Betreff für das Ticket an.

      Schritt 6. Stellen Sie alle Informationen zusammen und fügen Sie diese Informationen dem Abschnitt Beschreibung hinzu. Hierzu gehören beispielsweise folgende Angaben:

      • Datum, einschließlich Uhrzeit, an dem das Problem aufgetreten ist.
      • Häufigkeit des Problems (sporadisch oder konstant, zu einer bestimmten Tageszeit usw.)
      • Schritte zum Replizieren des Problems
      • Namen der betroffenen Server
      • Betroffene Rechenzentren
      • Alle zurückgegebenen Fehlernachrichten, sofern zutreffend
      • Relevante Serverprotokolle
      • Weitere hilfreiche Informationen

      Schritt 7. Fügen Sie direkt unter dem Abschnitt Beschreibung Anhänge, wie z. B. Protokolldateien, hinzu.

      Schritt 8. Wenn sich Ihre Bereitstellung auf eine neue Cloud bezieht, wählen Sie das Kontrollkästchen Bereitstellung aus.

      **Anmerkung: Die Schritte 9 und 10 können bei Problemstellungen, die nicht dringend sind, übersprungen werden. **

      Schritt 9. Wenn Ihr Problem dringend ist, können Sie das Kontrollkästchen Ticket als dringendes Ticket öffnen auswählen, um das Ticket zu eskalieren. Mit der Auswahl dieser Option geben Sie Ihr Einverständnis für die Zahlung einer Bearbeitungsgebühr, falls das Problem als nicht kritisch eingestuft wird. Zu den kritischen Situationen gehören Ausfälle oder Betriebsunterbrechungen.

      Schritt 10. Wenn Sie Ihr Ticket als dringendes Ticket öffnen möchten und mit der Zahlung einer Bearbeitungsgebühr einverstanden sind, falls das Problem als nicht kritisch eingestuft wird, wählen Sie das Kontrollkästchen Ich stimme zu aus.

      Schritt 11. Klicken Sie abschließend auf das Kontrollkästchen Ticket erstellen.

      Support-Ticket verwenden

      Sobald ein Ticket erstellt wurde, können Sie den Text zu diesem Ticket anzeigen, indem Sie in Ihrem Dashboard auf dieses Support-Ticket klicken. Der Tickettext kann Verweise auf andere Tickets sowie Chatverlaufsprotokolle und die Korrespondenz zu dem von Ihnen ausgewählten Ticket enthalten. Der Status Ihres Support-Tickets wird in der rechten oberen Ecke hervorgehoben. Es gibt Statusanzeiger für die Status “Offen”, “Beendigung anstehend”, “Geparkt” usw.

      Im Bereich auf der linken Seite der Anzeige können Sie ein weiteres neues Support-Ticket erstellen, ein von Ihnen ausgewähltes Ticket abonnieren (wenn Sie den zugehörigen Verlauf verfolgen möchten), Ihre eigenen Tickets anzeigen sowie alle Tickets anzeigen, die von Benutzern, die mit Ihnen das Konto gemeinsam nutzen, erstellt wurden.

      Wenn Sie eine Liste der Abonnenten für ein ausgewähltes Ticket anzeigen möchten, klicken Sie auf den Link Abonnenten.

      Wenn Sie einen schriftlichen Dialog in Bezug auf das Ticket mit einem der Experten des Bluemix Private Cloud-Support-Team wünschen, klicken Sie auf die grüne Schaltfläche Antwort veröffentlichen. Alternativ können Sie das Feature Live-Chat verwenden, indem Sie auf Chat klicken. Alle Live-Chat-Sitzungen werden aufgezeichnet und dem Ticket als Mitschrift hinzugefügt.

      Chat-Feature zum Chatten mit dem Bluemix Private Cloud-Support verwenden

      1. Klicken Sie in Ihrem Dashboard auf das Register Chat. Auf diese Weise werden Sie automatisch zu einem Chatroom mit Mitarbeitern des Support-Teams weitergeleitet.
      2. Im linken Bereich des Chatrooms sehen Sie, ob bereits ein Mitarbeiter des Support-Teams Ihrem Chatroom beigetreten ist.
      3. Sobald ein Mitarbeiter des Support-Teams anwesend ist, können Sie in der Chatleiste Nachrichten verfassen und senden. Wenn Sie während Ihres Chats eine neue Zeile einfügen möchten, drücken Sie die Tastenkombination Strg/Befehlstaste + Eingabetaste.
      4. Wenn Sie Ihr Support-Problem eskalieren müssen, rufen Sie die Nummer 1-800-613-4305 (USA) mit der Nebenstelle 1 an.

      Anmerkung: Jeder Kundenchatroom ist privat. Das bedeutet, dass nur die zu Ihrem Konto gehörenden Kontakte und die Mitarbeiter des Bluemix Private Cloud-Support-Teams auf Ihre Mitschriften zugreifen können. Es werden keine vertraulichen Daten (wie z. B. Kreditkartennummern oder private Schlüssel) persistent gespeichert.

      Abrechnungen verwalten

      Wenn Sie Ihre Verträge und monatlichen Abrechnungsberichte anzeigen möchten, klicken Sie im Box Panel-Dashboard auf das Register Konto und anschließend auf das Register Abrechnung.

      Wenn Sie darüber hinaus Unterstützung bei den Abrechnungen benötigen, wenden Sie sich bitte an das IBM Support Team, das Ihnen gern behilflich ist.
