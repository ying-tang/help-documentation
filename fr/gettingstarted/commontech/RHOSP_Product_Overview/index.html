      IBM Bluemix Private Cloud with Red Hat - Présentation du produit

      • by Leslie Lundquist ● 31 mars 2017

      • Tags:

        • getting started

        • getting started(initiation)

        • release(édition)

        • features(fonctionnalités)

        • overview(présentation)

      IBM Bluemix Private Cloud with Red Hat offre un cloud privé géré, déployé sur un système d’exploitation Red Hat Enterprise Linux (RHEL) 7.3. Il utilise les packages OpenStack en aval fournis par Red Hat OpenStack Platform (RHOSP) et basés sur RHOSP 10, qui utilise l’édition Newton d’OpenStack.

      Fonctionnalités

      L’édition 4.0.0.0r d’IBM Bluemix Private Cloud with Red Hat offre les fonctionnalités de cloud de haut niveau suivantes :

      • Cloud OpenStack, basé sur RHOSP 10 (c’est-à-dire l’édition Newton d’OpenStack)
      • Différents types de noeud de calcul adaptés à diverses charges de travail
      • Personnalisation de la zone de disponibilité Nova, qui permet de regrouper différents noeuds de calcul pour les placements de charge de travail
      • Possibilité de personnaliser le magasin d’images Glance en stockage d’objets, stockage Ceph ou système de fichiers local sur les contrôleurs dédiés
      • Prise en charge de Cinder avec le système de back end Ceph
      • Personnalisations du réseau, à savoir :
      • Réseau local virtuel Transit
      • Réseau privé virtuel IPSec ou OpenVPN
      • BYOIP dans le réseau superposé OpenStack
      • Accès au réseau privé entre deux comptes SoftLayer
      • Connectivité Direct Link SoftLayer
      • Equilibrage de charge en tant que service (LBaaS v2 avec le pilote HAProxy)
      • Gestion du cloud et de l’identité de l’utilisateur basée sur les rôles avec OpenStack Keystone
      • Services de stockage du système de back end Ceph, avec le stockage par blocs SSD disponible
      • Orchestration avec Heat et IBM Urban Code Deploy
      • Catalogue d’images publiques amorçables avec la fonction BYOL client

      Pour obtenir des informations générales sur les fonctionnalités fournies par IBM Bluemix Private Cloud, voir Présentation générale du produit.

      Expérience client

      IBM Bluemix Private Cloud with Red Hat offre la même expérience client qu’IBM Bluemix Private Cloud, à l’exception des deux options suivantes :

      Option IBM Bluemix Private Cloud with Red Hat IBM Bluemix Private Cloud
      Système d'exploitation RHEL 7.3 Ubuntu 14.04
      Edition d'OpenStack RHOSP 10 basé sur OpenStack Newton OpenStack Mitaka

      Etant donné que l’expérience client est similaire, vous pouvez consulter la documentation client sur notre site d’aide, même si elle n’est pas spécifiquement destinée aux clusters IBM Bluemix Private Cloud with Red Hat.

      Pour obtenir la nomenclature matérielle adaptée à votre environnement de cloud, contactez votre équipe de service. La configuration peut varier en fonction des emplacements de centre de données SoftLayer et de l’inventaire actuel.

      Box Panel

      La suite d’applications Box Panel offre une vue d’ensemble unique des actifs déployés dans votre environnement de cloud.

      Box Panel est conçu pour déployer et exploiter une infrastructure de cloud privé tout au long de son cycle de vie. Il permet notamment de gérer les logiciels et les services, de la configuration de l’armoire à la facturation, et permet d’interagir avec le service clients.

      La suite Box Panel inclut plusieurs applications qui gèrent les différents domaines de sa fonctionnalité :

      • Box Panel — entrepôt d’informations qui permet de gérer les machines, les diagrammes de montage en armoire, les adresses IP, les déploiements réseau et les informations connexes.
      • Box Panel Auth — application qui autorise les utilisateurs et permet de gérer les utilisateurs et les projets par l’intermédiaire du contrôle d’accès basé sur les rôles (RBAC).
      • Box Panel Support — application qui permet de gérer les tickets de demande de service client et les fonctions de support connexes.
      • Box Panel Chat — application qui permet aux clients d’interagir via la discussion en ligne.
      • Box Panel Messenger Service — application qui permet d’interagir avec le contrôleur de site et Ursula pour offrir des fonctions de déploiement automatisé, d’extension et de mise à jour par le biais d’une architecture basée sur les services et les employés.
