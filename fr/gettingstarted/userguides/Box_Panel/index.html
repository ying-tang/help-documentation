      Initiation à Box Panel

      L’interface Box Panel offre une vue d’ensemble unique des actifs déployés dans vos environnements IBM Bluemix Private Cloud et IBM Bluemix Private Cloud Local. Son accès en libre-service vous permet de créer des tickets de demande de service, d’optimiser l’authentification centrale des connexions et des droits utilisateur, de surveiller les différentes fonctions et d’afficher vos factures facilement.

      Le présent guide d’utilisation offre un aperçu de bout en bout, de sorte que vous puissiez commencer à utiliser Box Panel pour surveiller et contrôler votre environnement IBM Bluemix Cloud. Le présent document s’applique à Box Panel version 3.8.3 et ultérieure.

      Connexion

      Pour débuter avec Box Panel, connectez-vous.

      1. Accédez à l’URL https://boxpanel.bluebox.net pour être redirigé vers la page de connexion.
      2. Connectez-vous à votre compte Box Panel à l’aide des données d’identification de connexion fournies par l’équipe de support IBM Bluemix Private Cloud. Si vous êtes un contact secondaire et ne disposez pas encore d’un accès, vous pouvez être ajouté par un administrateur. Pour plus de détails, voir la section Gestion des utilisateurs et des projets.
      3. Si vous ne disposez pas encore d’un compte Box Panel, consultez la section Facturation pour obtenir des instructions sur l’obtention d’un compte.
      4. Une fois connecté, vous serez redirigé automatiquement vers le tableau de bord Box Panel.

      Obtention de vos données d’identification à partir de Lock Box

      Vous recevrez dans un premier temps les données d’identification cloud_admin de votre cloud privé (cluster OpenStack) par l’intermédiaire de Box Panel Lockbox. Vos données d’identification devraient ressembler à ce qui suit :

      Pour commencer, accédez à l’URL fournie dans Lock Box. Utilisez ensuite votre nom d’utilisateur spécifié et votre mot de passe spécifié pour vous connecter.

      Navigation dans le tableau de bord Box Panel

      Le tableau de bord Box Panel offre une vue d’ensemble des actifs de toutes vos instances IBM Bluemix Private Cloud et IBM Bluemix Private Cloud Local sur un seul écran.

      La partie supérieure du tableau de bord affiche l’infrastructure actuellement utilisée ainsi que la liste des hôtes.

      La partie droite contient des détails supplémentaires sur votre résumé de compte, notamment vos Messages Lock Box, votre Mot de passe verbal, le Quota de bande passante et le Quota de sauvegarde.

      La partie supérieure droite du tableau de bord affiche les onglets et options de menu suivants.

      Onglets :

      • Support : permet d’afficher les tickets existants et d’en créer de nouveaux. Pour plus d’informations, voir Obtention d’aide.
      • Dialoguez en ligne : permet de discuter en ligne avec le personnel d’assistance d’IBM Bluemix Private Cloud. Voir la section “Utilisation de la fonction Dialoguez en ligne pour discuter avec le support Bluemix Private Cloud” dans Obtention d’aide.
      • Messagerie Web : permet d’utiliser la messagerie Web hébergée sur https://mail.blueboxgrid.com/.
      • Moi : permet d’afficher ou de mettre à jour votre profil, ou encore de changer votre mot de passe.

      Menus :

      • Changer de compte : si vous avez accès à plusieurs instances d’IBM Bluemix Private Cloud, vous pouvez utiliser cette option pour changer de compte.
      • Comptes : affichez les contacts de votre compte, les messages Lock Box, les informations de facturation, la procédure de transmission au niveau supérieur et les informations relatives à votre compte.
      • Services : affichez tous les services Cloud Blue Box et Hébergement géré disponibles pour votre compte, y compris les Serveurs, les Images de cloud, le DNS, les Graphiques de bande passante, les Données de bande passante et New Relic. Les options peuvent différer en fonction des types de client.
      • Docs : affichez la documentation d’aide. Pour plus d’informations, voir la section Documentation d’aide.

      Gestion de vos contacts

      Pour afficher et mettre à jour vos informations de contact client dans Box Panel, procédez comme suit :

      • Connectez-vous à Box Panel.
      • A partir du menu Compte, sélectionnez Contacts du compte pour ouvrir la page Contacts du compte.

      • Tous les contacts autorisés à accéder à votre compte sont répertoriés sous la section Contacts autorisés. Le nom et l’adresse e-mail de chaque personne sont répertoriés, ainsi que trois rôles du contact (remarque : un contact peut avoir l’un de ces rôles ou plus) :

        • Les contacts Admin (A) peuvent commander ou supprimer des services, mais également gérer d’autres contacts.
        • Les contacts Facturation (F) peuvent afficher et régler des factures.
        • Les contacts Technique (T) peuvent ouvrir des tickets et contacter l’équipe de support Bluemix Private Cloud pour obtenir de l’aide.
        • Le contact Principal (P), qui est unique, a le contrôle administratif sur le compte. Il ne peut être modifié que par le personnel d’assistance d’IBM Bluemix Private Cloud.

      Pour ajouter un nouveau contact, cliquez sur le bouton Ajouter un contact. Le formulaire Ajouter un contact s’affiche. Vous pouvez y saisir le nom du contact, son adresse e-mail, son numéro de téléphone et le(s) rôle(s) à lui attribuer. Une fois terminé, cliquez sur le bouton Enregistrer les détails pour ajouter le contact.

      Pour supprimer un contact existant, cliquez sur la croix (X) figurant sur la ligne où apparaît son nom dans la liste Contacts autorisés.

      Notez que le contact Principal ne peut pas être supprimé. Contactez le support IBM Bluemix Private Cloud pour obtenir de l’aide sur la modification du contact Principal.

      Utilisation des images de cloud

      La page Images de cloud de Box Panel est accessible aux clients qui disposent au moins d’un cloud et qui sont contacts client de type Principal ou Technique. Si vous obtenez le statut d’utilisateur, la section Services contiendra un lien Images de cloud avec un accès à la page.

      ![ (https://cloud.githubusercontent.com/assets/17212946/15030057/fb4cdeec-1215-11e6-9d3e-60fa8d7953a9.png)

      Chaque image apparaît sous forme de carte. Les cartes sont regroupées par système d’exploitation.

      Pour télécharger une image de cloud :

      1. Survolez la carte associée à l’image que vous voulez télécharger.
      2. Cliquez sur la carte. Une fenêtre modale s’ouvre, contenant des liens de téléchargement vers l’image et la somme de contrôle.
      3. Cliquez sur le bouton Copier à gauche de l’URL de l’image. L’URL est ajoutée à votre presse-papiers.
      4. Cette procédure peut également être effectuée à l’aide des API OpenStack. Pour obtenir des instructions, consultez le document principal relatif aux images de cloud.

      Documentation d’aide

      A partir de votre tableau de bord Box Panel, un lien intitulé Docs apparaît sur la barre de navigation (lien le plus à droite). Si vous cliquez sur ce lien, un menu déroulant s’affiche, contenant les options OpenStack, Cloud Object Storage ou Hébergement géré. Pour afficher la documentation d’aide destinée aux clients d’IBM Bluemix Private Cloud, sélectionnez OpenStack. La page Aide s’affiche. Vous pouvez parcourir la documentation par catégorie, ou effectuer une recherche à l’aide de la barre de recherche. En survolant l’une des icônes, vous pouvez afficher la liste des articles contenus dans cette zone de rubrique.

      Obtention d’aide

      Remarque : pour obtenir de l’aide immédiatement, vous pouvez appeler le support Bluemix Private Cloud au 1-800-613-4305 ou nous envoyer un e-mail à l’adresse support@bluebox.net.

      Vous souhaiterez, dans la plupart des cas, ouvrir un ticket de demande de service pour obtenir de l’aide. En utilisant un ticket, vous disposerez d’un enregistrement compilant le problème et les solutions choisies pour résoudre votre problème, qui sera complété avec les journaux de vos sessions de discussion avec l’équipe du service clients.

      Pour créer un ticket :

      Etape 1. Cliquez sur Nouveau ticket sur la page Support.

      Etape 2. Dans la fenêtre modale, votre nom apparaît comme Créateur du ticket.

      Etape 3. Sélectionnez une catégorie (Support ou Facturation). Vérifiez la liste des utilisateurs qui ont accès à votre ticket.

      Etape 4. Copiez des utilisateurs supplémentaires dans la zone Cc autres utilisateurs pour les informer du ticket et qu’ils puissent suivre son statut.

      Etape 5. Ajoutez un objet descriptif au ticket.

      Etape 6. Collectez et ajoutez toutes les informations utiles à la section Description, telles que :

      • Date et heure de début du problème
      • Fréquence du problème (permanente ou constante, à une heure spécifique de la journée, etc.)
      • Etapes permettant de reproduire le problème
      • Nom(s) de(s) serveur(s) concerné(s)
      • Centre(s) de données concerné(s)
      • Eventuels messages d’erreur reçus
      • Journaux serveur applicables
      • Toute information complémentaire pouvant être utile

      Etape 7. Insérez des pièces jointes, telles que des fichiers journaux, sous la section Description.

      Etape 8. Si votre déploiement cible un nouveau cloud, cochez la case Déploiement.

      ![ (https://cloud.githubusercontent.com/assets/17212946/15032633/850d3d4a-1228-11e6-9b37-c20e5d32d9b9.png)

      Remarque : vous pouvez ignorer les étapes 9 et 10 si votre problème n’est pas urgent.

      Etape 9. Si votre problème est urgent, vous pouvez cocher la case Ouvrir ce ticket avec le statut Urgent pour faire remonter le ticket. Vous recevrez un accusé de réception indiquant que vous consentez à payer des frais de consultation si le problème n’est pas critique (un problème critique concerne les situations de panne par exemple).

      Etape 10. Si vous souhaitez ouvrir votre ticket avec un statut Urgent et consentez à payer des frais de consultation si votre problème n’est pas critique, cochez la case J’accepte.

      Etape 11. Cliquez enfin sur le bouton Créer un ticket.

      Utilisation de votre ticket de demande de service

      Une fois votre ticket créé, vous pouvez consulter le texte qui lui est associé en cliquant sur ce ticket de demande de service à partir de votre tableau de bord. Le texte du ticket peut inclure des références à d’autres tickets, ainsi que l’historique des discussions et toute correspondance relative au ticket sélectionné. Le statut de votre ticket de demande de service est mis en évidence dans le coin supérieur droit. Les indicateurs de statut sont : OUVERT, EN ATTENTE D’ACHEVEMENT, PARQUE, etc.

      A partir du panneau situé dans la partie gauche de l’écran, vous pouvez sélectionner les options permettant de créer un autre ticket de demande de service, de s’abonner au ticket sélectionné (pour suivre son statut), de visualiser vos tickets et d’afficher tous les tickets soumis par les utilisateurs partagés sur votre compte.

      Pour afficher la liste des abonnés à un ticket sélectionné, cliquez sur le lien Abonnés.

      Pour discuter par écrit avec l’un des spécialistes de notre équipe de support Bluemix Private Cloud à propos du contenu de votre ticket, vous pouvez cliquer sur le bouton Publier la réponse vert, ou utiliser la fonction de discussion en ligne en cliquant sur Dialoguez en ligne. Toutes les sessions de discussion en ligne sont enregistrées et ajoutées au ticket sous forme de retranscription.

      Utilisation de la fonction Dialoguez en ligne pour discuter avec le support Bluemix Private Cloud

      1. Cliquez sur l’onglet Dialoguez en ligne à partir de votre tableau de bord. Vous êtes redirigé automatiquement vers un salon de discussion afin de dialoguer avec le personnel d’assistance.
      2. A partir d’ici, vous pouvez voir les membres du personnel d’assistance rejoindre votre conversation dans la partie gauche du salon de discussion.
      3. Une fois qu’un membre du support a rejoint le salon, vous pouvez rédiger et envoyer des messages à partir de la barre de discussion. Pour passer à la ligne au cours de la discussion, appuyez sur Ctrl + Entrée.
      4. Pour faire remonter votre problème, appelez le numéro 1-800-613-4305 ext 1.

      Remarque : chaque salon de discussion est privé, ce qui signifie que seuls les contacts définis dans votre compte client et le personnel d’assistance Bluemix Private Cloud ont accès à votre retranscription. Aucune donnée sensible (comme les numéros de carte de crédit ou les clés privées) n’est conservée.

      Gestion de la facturation

      Pour afficher vos contrats et vos rapports de facturation mensuels, cliquez sur l’onglet Compte dans le tableau de bord Box Panel, puis cliquez sur l’onglet Facturation.

      Pour obtenir de l’aide sur la facturation, contactez l’équipe de support IBM, nous serons heureux de vous aider.
