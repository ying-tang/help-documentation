      IBM Bluemix Private Cloud カスタマー

      リリース情報 3.1.0

      2016 年 10 月 14 日

      この文書では、3.1.0 IBM Bluemix Private Cloud および IBM Bluemix Private Cloud Local オファリングの基本的な説明をします。各セクションで詳細な資料へのリンクを提供しています。

      最新リリースの要約

      IBM Bluemix Private Cloud および IBM Bluemix Private Cloud Local のリリース 3.1.0 には、いくつかの新機能およびパフォーマンスの改善が含まれています。これは引き続き、OpenStack Mitaka リリースに基づきます。

      すべての IBM Bluemix Private Cloud には、3.1.0 の以下の新機能があります。

      1.) フェデレーテッド Keystone 顧客 ID プロバイダー (IdP): お客様は、Security Assertion Markup Language (SAML) や OpenIDConnect などの既存の ID インフラストラクチャー・フェデレーション・プロトコルを使用して、自分の ID ディレクトリー (LDAP、RADIUS、Active Directory など) に格納された ID を使用できるようになりました。お客様が管理する単一の資格情報を使用して、複数の OpenStack クラウドを管理することができます。

      以下は、関連する資料の一部です。

      • K2K フェデレーション
      • SAML フェデレーション
      • OIDC フェデレーション

      2.) ストレージ・プール: 様々なストレージ・パフォーマンス要件を持つお客様による、同じクラウド内にある高性能の SSD およびコスト効果の高いハイブリッド・ストレージ・プールへのワークロードの最適な送信が可能になりました。新規デプロイメント用の最小の OSD ノード数は 3 のままです。ハイブリッド拡張については最低 3 ノードを追加する必要があります。それ以降は、一度に 1 ノードずつストレージを拡張できます。

      以下は、関連する資料の一部です。

      • サポートされるボリューム・タイプ
      • Horizon からのボリュームの作成
      • コマンド・ラインからのボリュームの作成

      3.) ストレージのパフォーマンス改善: Ceph デプロイメントが、Jemalloc を使用可能な Jewel リリースにアップグレードされ、パフォーマンスが改善されています。

      IBM Bluemix Private Cloud イメージの更新

      IBM Bluemix Private Cloud イメージの更新: 使用可能なクラウド・イメージのライブラリーが更新されました。最新の情報は、Cloud イメージ・ユーザー・ガイドを参照してください。

      IBM Bluemix Private Cloud 3.1.0 の機能 (IBM クラウド・データ・センター内):

      • Bluemix Private Cloud (IBM クラウド・データ・センター内) は、ISO27001 および 270018 に準拠し、SOC 2 および SOC 3 にも準拠しています。

      IBM Bluemix Private Cloud Local 3.1.0 の機能:

      • Lenovo のサポート: 現在の Cisco UCS プラットフォームに加えて、Lenovo でキュレーションされた部品表がサポートされるようになりました。
      • Bluemix Local システムのサポート: IBM Bluemix Private Cloud Local (PCaaS) Private Cloud as a Service は、Bluemix Local システムでサポートされるようになりました。

      本リリースの既知の制限事項

      Horizon では、フェデレーテッド・ユーザーがパスワードを変更できるページがあります。ユーザーの ID 情報が格納されていないため、パスワードを変更しようとしたときにエラーが発生します。(障害 #236958)

      Horizon で、グループを通じて project_admin アクセス権が付与されているユーザーの場合、「ID (Identity)」->「プロジェクト (Projects)」でユーザーが表示されません。(障害 #237038)

      LBaaS の制約に関する具体的な情報については、LBaaS v2 の制限事項を参照してください。

      さらに、ユーザー向け資料については、オンラインで常に利用可能な help.blueboxcloud.com の カスタマー・ヘルプのメイン・ページを参照してください。

      IBM Bluemix Private Cloud の新規ユーザー用に、IBM Bluemix Private Cloud バージョン 2.0 以降の一般的な製品概要がが提供されています。
