      OpenVPN サーバーへの接続

      • by Jason Kennedy ● January 7th, 2016

      • Tags:

        • はじめに

        • openVPN

        • 接続

      OpenStack クラスターに OpenVPN サーバーが作成されたため、次はサーバーに接続します。

      接続は、.ovpn ファイルに付けた名前で呼ばれます。この例では、ファイルの名前は Client1.ovpn であるため、接続の名前は Client1 になります。

      Windows

      インストール

      Windows 用の OpenVPN クライアント・アプリケーションは OpenVPN のダウンロード・ページから見つけることができます。Windows のバージョンに適したインストーラー・バージョンを選択してください。

      注: OpenVPN をインストールするには管理特権が必要です。

      OpenVPN をインストールした後で、一元化された DigitalOcean.ovpn プロファイルを次の場所にコピーします。

      C:\Program Files\OpenVPN\config

      OpenVPN を起動すると、OpenVPN は自動的にプロファイルを確認して利用できるようにします。

      OpenVPN を使用するときは、管理用アカウントであっても、毎回管理者として実行する必要があります。毎回 VPN を使用するたびに右クリックし「管理者として実行」を選択しなくてもいいように、この値を事前設定することができます。ただし、この事前設定は管理用アカウントから行う必要があります。この制限は、標準ユーザーが OpenVPN を使用するには管理者のパスワードを入力する必要がある、ということも意味しています。一方で、クライアントの OpenVPN に管理権限がなければ標準ユーザーはサーバーに適切に接続できないため、権限の昇格が必要になります。

      OpenVPN アプリケーションが常に管理者として実行されるようにするには、ショートカット・アイコンを右クリックして、「プロパティ」に移動します。「互換性」タブの一番下にある「すべてのユーザーの設定を変更」ボタンをクリックします。新しいウィンドウで、「管理者としてこのプログラムを実行する」にチェック・マークを付けます。

      接続

      OpenVPN GUI を起動するたび、Windows はプログラムがコンピューターに対して変更を行うことを許可するかどうかを尋ねます。「はい」をクリックします。OpenVPN クライアント・アプリケーションを起動しただけでは、アプレットがシステム・トレイに置かれて VPN が必要に応じて接続および切断できるようになるだけで、VPN 接続は実際には行われません。

      OpenVPN が開始されたら、システム・トレイ・アプレットに移動し、OpenVPN アプレット・アイコンを右クリックして、接続を開始します。これによりコンテキスト・メニューが開きます。メニューの上部にある「Client1」(これは Client1.ovpn プロファイルです) を選択し、「接続」を選択します。

      接続が確立されている間のログ出力を示す状況ウィンドウが開き、クライアントが接続されるとメッセージが表示されます。

      同じ方法で VPN から切断します。システム・トレイ・アプレットに移動し、OpenVPN アプレット・アイコンを右クリックしてクライアント・プロファイルを選択し、「切断」をクリックします。

      OS X

      インストール

      Tunnelblick は、Mac OS X 用のオープン・ソースの無料 OpenVPN クライアントです。Tunnelblick のダウンロード・ページから最新のディスク・イメージをダウンロードできます。ダウンロードした .dmg ファイルをダブルクリックし、プロンプトに従ってインストールします。

      インストール・プロセスの終盤に、Tunnelblick によって構成ファイルがあるかどうかを尋ねられます。より簡単な方法として、「いいえ (No)」と回答し Tunnelblick にプロセスを完了させることをお勧めします。Finder ウィンドウを開き、DigitalOcean.ovpn をダブルクリックします。Tunnelblick がクライアント・プロファイルをインストールします。管理特権が必要です。

      接続

      「アプリケーション」フォルダーの Tunnelblick をダブルクリックして、Tunnelblick を起動します。Tunnelblick が起動されると、接続を制御するための Tunnelblick アイコンがメニュー・バーの画面の右上に表示されます。アイコンをクリックしてから、VPN 接続を開始するための「接続 (Connect)」メニュー項目を選択します。「Client1」接続を選択します。

      iOS

      インストール

      iTunes App Store で OpenVPN Connect (正規の iOS OpenVPN クライアント・アプリケーション) を検索してインストールします。iOS クライアント・プロファイルをデバイスに転送するには、デバイスをコンピューターに直接接続します。

      iTunes を使用した転送の概要を以下に説明します。コンピューターで iTunes を開き、「iPhone」>「App」をクリックします。「ファイル共有」セクションの下部までスクロールダウンし、OpenVPN アプリをクリックします。右側のブランク・ウィンドウ「OpenVPN の書類」は、ファイル共有のためのウィンドウです。.ovpn ファイルを「OpenVPN の書類」ウィンドウにドラッグします。

      次に、iPhone 上で OpenVPN アプリを起動します。新規プロファイルを今すぐインポートできることを示す通知が表示されます。緑の正符号 (+) をタップしてインポートします。

      接続

      OpenVPN で新規プロファイルを使用する準備ができました。「接続 (Connect)」ボタンを「オン (On)」の位置にスライドして接続を開始します。同じボタンを「オフ (Off)」にスライドすると切断されます。

      注: 「設定」にある VPN スイッチは VPN を接続するために使用できません。この操作を試すと、OpenVPN アプリのみを使用して接続するように指示する通知を受け取ります。

      Android

      インストール

      Google Play ストアを開きます。Android OpenVPN Connect (正規の Android OpenVPN クライアント・アプリケーション) を検索してインストールします。

      Android デバイスをコンピューターに USB で接続してファイルをコピーすると、.ovpn プロファイルを転送できます。あるいは、SD カード・リーダーがある場合は、デバイスの SD カードを取り外して、プロファイルをカードにコピーし、その後カードを Android デバイスに戻すこともできます。

      OpenVPN アプリを開始し、メニューをタップしてプロファイルをインポートします。

      次に、保存されたプロファイルの場所にナビゲートし (スクリーン・ショットでは /sdcard/Download/ を使用)、ファイルを選択します。プロファイルがインポートされたことがアプリにメモされます。

      接続

      接続するには、「接続 (Connect)」ボタンをタップします。OpenVPN アプリケーションを信頼するかどうかを尋ねられます。「OK」を選択して、接続を開始します。VPN から切断するには、OpenVPN アプリに戻って、*「切断 (Disconnect)」を選択します。

      VPN 接続のテスト

      すべてインストールされたら、単純なチェックによりすべてが正常に機能していることを確認できます。VPN 接続が有効でない状態で、ブラウザーを開いて DNSLeakTest に移動します。

      このサイトでは、インターネット・サービス・プロバイダーによって割り当てられた IP アドレスと、世界のユーザーからどのように見えるかが返されます。この Web サイトで DNS 設定を確認するには、「Extended Test」をクリックすると、使用中の DNS サーバーが通知されます。

      次に、OpenVPN クライアントを Droplet の VPN に接続し、ブラウザーを最新表示します。VPN サーバーのまったく別の IP アドレスが表示されるはずです。現時点では世界のユーザーからこのように見えます。繰り返しになりますが、DNSLeakTest Extended Test は、DNS 設定を検査し、使用する VPN によってプッシュされた DNS リゾルバーを使用していることを確認します。

      おめでとうございます。これで、自分の ID、ロケーション、およびトラフィックを盗み見や検閲から保護しつつ、インターネット上で安全に通信を行うことができます。
