      Box Panel ユーザー・ガイド

      • by Jill Tempelmeyer and Leslie Lundquist ● April 14, 2016

      Box Panel の概要

      Box Panel インターフェースを使用すると、IBM Bluemix Private Cloud および IBM Bluemix Private Cloud Local の環境にデプロイされている資産を、1 カ所にまとめて表示することができます。ユーザーは、そのセルフサービスのアクセス機能を使用することで、サポート・チケットの作成、ログインやユーザー権限に対する中央認証の活用、各種機能のモニター、および請求書の表示を、簡単な操作で行うことができます。

      このユーザー・ガイドでは、ユーザーが Box Panel を使って IBM Bluemix Cloud 環境のモニターと制御を始めることができるように、その概要を順を追って説明します。本資料は Box Panel バージョン 3.8.3 以降に適用されます。

      ログイン

      Box Panel の使用を開始するには、Box Panel にログインします。

      1. URL https://boxpanel.bluebox.net にアクセスすると、ログイン・ページに誘導されます。
      2. IBM Bluemix Private Cloud サポート・チームから提供されたログイン資格情報を使用して、Box Panel アカウントにログインします。ユーザーが 2 次連絡先に該当し、まだアクセス権限を保持していない場合は、管理者がそのユーザーを追加できます。詳しくは、『ユーザーおよびプロジェクトの管理』のセクションを参照してください。
      3. Box Panel アカウントをまだ取得していない場合は、請求のセクションでアカウント取得方法についての説明を確認してください。
      4. ログインが完了すると、Box Panel ダッシュボードに自動的に誘導されます。

      Lock Box からの資格情報の取得

      最初に、Box Panel Lock Box を通じて、プライベート・クラウド (OpenStack クラスター) の cloud_admin 資格情報を受け取ります。この資格情報は以下のようなものです。

      まず始めに、Lock Box で示された URL にナビゲートします。次に、指定したユーザー名と指定したパスワードを使用してログインします。

      Box Panel ダッシュボードのナビゲート

      Box Panel ダッシュボードから、IBM Bluemix Private Cloud および IBM Bluemix Private Cloud Local のすべてのインスタンスを 1 画面にまとめた資産ビューを表示できます。

      ダッシュボードの最上部には、現在使用されているインフラストラクチャーが、ホストのリストともに表示されます。

      右側には、ユーザーの利用料金の要約に関連する追加の詳細情報が表示されます (Lock Box メッセージ、口頭パスワード、帯域幅の割り当て量、バックアップ割り当て量など)。

      右上には、以下に示すタブとメニュー・オプションが表示されます。

      タブ:

      • サポート: 既存のチケットの表示、および新規チケットの作成を行います。詳しくは、『サポートの利用』を参照してください。
      • チャット: IBM Bluemix Private Cloud サポート・スタッフとライブ・チャットを行います。『サポートの利用』のセクション『チャット機能を使用した Bluemix Private Cloud サポートとのチャット』を参照してください。
      • Webmail: https://mail.blueboxgrid.com/ にホストされている Web メールを使用します。
      • 自分: 自分のプロファイルの表示または更新、パスワードの変更を行います。

      メニュー:

      • アカウントの切り替え: 複数の IBM Bluemix Private Cloud インスタンスへのアクセス権限を保持している場合は、このオプションを使用して別のアカウントに切り替えることができます。
      • アカウント: アカウントの連絡先、Lock Box メッセージ、請求情報、エスカレーション手順、およびアカウント情報を表示します。
      • サービス: アカウントが使用できる Blue Box クラウドおよびマネージド・ホスティングのすべてのサービスが表示されます (サーバー、クラウド・イメージ、DNS、帯域幅グラフ、帯域幅データ、New Relic など)。オプションはお客様のタイプによって異なります。
      • 文書: ヘルプ文書を表示します。詳しくは、『ヘルプ文書の表示』を参照してください。

      連絡先の管理

      Box Panel でお客様連絡先情報の表示や更新を行うには、以下の手順を実行します。

      • Box Panel にログインします。
      • 「アカウント」メニューで「アカウント・コンタクト」を選択して、「アカウント・コンタクト」ページを開きます。

      • アカウントへのアクセス権限を持つすべての連絡先が、「権限を有する担当者」セクションにリストされます。各連絡先の名前と E メール・アドレスが、連絡先の 3 つの役割とともにリストされます (1 件の連絡先に、以下に示す連絡先役割のうち 1 つ以上が割り当てられることがあります)。

        • 管理者 (A) 連絡先。サービスの依頼または削除のほか、他の連絡先の管理も行うことができます
        • 請求 (B) 連絡先。請求書の表示と支払いを行うことができます
        • 技術 (T) 連絡先。チケットのオープンと、Bluemix Private Cloud サポート・チームへの支援の依頼を行うことができます
        • 主要 (P) 連絡先。この担当者は 1 人だけであり、アカウントに対する管理制御権限を持っています。主要連絡先を変更できるのは、IBM Bluemix Private Cloud サポート・スタッフだけです。

      新規連絡先を追加するには、「連絡先の追加」ボタンをクリックします。「連絡先の追加」フォームが表示され、連絡先の名前、E メール・アドレス、電話番号、およびその連絡先の所属先となる役割を入力できます。入力が完了したら、「詳細の保存」ボタンをクリックして連絡先を追加します。

      既存の連絡先を削除するには、「権限を有する担当者」リストで、その連絡先名が記載された行の「X」をクリックします。

      主要連絡先は削除できません。主要連絡先を変更するために支援が必要な場合は、IBM Bluemix Private Cloud サポートにご連絡ください。

      クラウド・イメージの操作

      Box Panel の「クラウド・イメージ」ページは、1 つ以上のクラウドを保持しており、「主要」または「技術」のお客様連絡先に該当するユーザーが使用できます。ユーザーの資格を有している場合は、このページにアクセスするための「クラウド・イメージ」リンクが「サービス」のナビゲーションに表示されます。

      各イメージはカードとして表示されます。カードはオペレーティング・システムごとにグループ化されています。

      クラウド・イメージをダウンロードするには、以下の手順を実行します。

      1. ダウンロードしたいイメージに関連付けられたカードの上にマウスを移動します。
      2. カードをクリックします。これにより、そのイメージとチェックサムのダウンロード・リンクを示すモーダル・ウィンドウが開きます。
      3. イメージ URL の左側の「コピー (Copy)」ボタンをクリックします。これにより、URL がクリップボードに追加されます。
      4. この手順は OpenStack API を使用する方法でも実行できます。その方法については、クラウド・イメージに関する主要文書を参照してください。

      ヘルプ文書の表示

      Box Panel ダッシュボードのナビゲーション・バーに、「文書」というリンク (右端) があります。このリンクを選択するとドロップダウン・メニューが表示され、「OpenStack」、「クラウド・オブジェクト・ストレージ」、および「マネージド・ホスティング」の各オプションが示されます。IBM Bluemix Private Cloud のお客様向けヘルプ文書を表示するには、「OpenStack」を選択します。これにより、ヘルプ・ページが表示されます。カテゴリー別に文書を参照することも、検索バーを使用して検索を実行することもできます。いずれかのアイコンの上にマウスを移動すると、そのトピックの領域に属する記事のリストが表示されます。

      サポートの利用

      注: すぐにサポートを受けたい場合は、Bluemix Private Cloud サポートにお電話いただくか (1-800-613-4305)、E メールでご連絡ください (support@bluebox.net)。

      多くの場合、サポートを利用するにはサポート・チケットをオープンします。チケットを利用することで、問題とその問題に対処するために取られた解決策を記録できるほか、お客様サポート・チームとのチャット・セッションのログを添付することができます。

      チケットを作成するには、以下の手順を実行します。

      ステップ 1. 「サポート」ページの「新規チケット」をクリックします。

      ステップ 2. モーダル・ウィンドウに、ユーザーの名前が「チケット作成者 (Ticket Creator)」として表示されます。

      ステップ 3. カテゴリー (「サポート」または「請求」) を選択します。このチケットへのアクセス権限を付与するユーザーのリストを確認します。

      ステップ 4. 別のユーザーにチケットについて通知し、その状況を追跡するよう促すには、「別のユーザーに CC」フィールドにそのユーザーをコピーします。

      ステップ 5. チケットについて説明する件名を追加します。

      ステップ 6. 有用な情報を収集して、「説明」セクションにすべて追加します。以下のような情報を追加します。

      • 問題が発生した日時
      • 問題の頻度 (常時なのか一定の頻度なのか、特定の時刻に発生するのかなど)
      • 問題を再現するために実行した手順
      • 影響を受けるサーバーの名前
      • 影響を受けるデータ・センター
      • 戻されるエラー・メッセージ
      • 該当するサーバー・ログ
      • その他の有用な情報

      ステップ 7. 「説明」セクションのすぐ下で、ログ・ファイルなどの添付ファイルを追加します。

      ステップ 8. デプロイメントが新規クラウド用である場合は、「デプロイメント (Deployment)」ボックスにチェック・マークを付けます。

      注: 緊急度の低い問題の場合は、ステップ 9 と 10 を省略できます。

      ステップ 9. 問題の緊急度が高い場合は、「このチケットを緊急としてオープン」ボックスにチェック・マークを付けて、チケットをエスカレートできます。これにより、問題が重要でない場合に相談料を支払うことに同意するかどうかの確認を求めるプロンプトが出されます (重要な項目とは障害が発生するような状態を指します)。

      ステップ 10. チケットを緊急としてオープンしたいと考えており、重要でない項目に相談料を支払うことに同意する場合は、「同意する」ボックスにチェック・マークを付けます。

      ステップ 11. 最後に、「チケットの作成」ボックスをクリックします。

      サポート・チケットの使用

      チケットの作成後は、ダッシュボードからそのサポート・チケットをクリックすることで、チケットに関連付けられたテキストをいつでも確認できます。チケットのテキストには、他のチケットへの参照や、チャット履歴、選択したチケットに関連する通信が含まれる場合があります。サポート・チケットの状況は右上隅に強調表示されます。状況標識には、「オープン」、「完了保留 (PENDING COMPLETION)」、「保留」などがあります。

      画面左側のパネルからオプションを選択することで、別の新規サポート・チケットの作成、選択したチケットのサブスクライブ (その状況をフォローしたい場合)、チケットの表示、およびアカウントの共有ユーザーが送信したすべてのチケットの表示を行うことができます。

      選択したチケットのサブスクライバーのリストを表示するには、「サブスクライバー」リンクをクリックします。

      このチケットのトピックについて、Bluemix Private Cloud サポート・チームの専門家と書面で対話するには、緑色の「返信の投稿」ボタンをクリックするか、「チャット」ボタンをクリックしてライブ・チャット機能を使用できます。ライブ・チャットのセッションはすべて記録され、トランスクリプトとしてチケットに追加されます。

      チャット機能を使用した Bluemix Private Cloud サポートとのチャット

      1. ダッシュボードの「チャット」タブをクリックします。これにより、サポート・スタッフとのチャット・ルームに自動的に誘導されます。
      2. この時点から、チャット・ルームの左側で、サポート・スタッフのメンバーが会話に参加したかどうかを確認できます。
      3. サポート・メンバーが参加したら、チャット・バーでメッセージを作成して送信できます。チャットの途中で改行を入力する必要がある場合は、Ctrl/Command + Enter を押します。
      4. サポートの問題をエスカレートする必要がある場合は、1-800-613-4305 内線 1 にお電話ください。

      注: お客様の各チャット・ルームは、他には公開されません。つまり、お客様アカウント内の連絡先と Bluemix Private Cloud サポート・スタッフだけが、トランスクリプトにアクセスできます。また、機密データ (クレジット・カード番号や秘密鍵など) が記録に残ることはありません。

      請求書の管理

      契約や月次請求レポートを表示するには、Box Panel ダッシュボードの「アカウント」タブをクリックして、「請求」タブをクリックします。

      ここに表示される請求情報以外の情報が必要な場合は、IBM サポート・チームにお問い合わせいただければ、いつでもお答えいたします。
